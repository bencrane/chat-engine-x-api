# RudderStack Quickstart

Get started with the RudderStack platform and its features.

* * *

  * __2 minute read

  * 


The easiest way to get started with RudderStack is to sign up for [RudderStack Cloud](<https://app.rudderstack.com/signup/>) and set up a [data pipeline](<https://www.rudderstack.com/docs/data-pipelines/overview/>) to collect data from a variety of cloud and warehouse sources, then automatically route them to the downstream tools in your stack.

RudderStack offers the following data pipelines:

  * **Event Stream** : With over 200 pre-built integrations, including 16 SDKs, you can use RudderStack’s [Event Stream](<https://www.rudderstack.com/docs/data-pipelines/event-stream/>) feature to stream data directly to your warehouse and other downstream tools in no time.


> ![info](/docs/images/info.svg)
> 
> See the [Event Stream Quickstart](<https://www.rudderstack.com/docs/data-pipelines/event-stream/quickstart/>) guide to set up a Event Stream pipeline in less than 15 minutes.

  * **Reverse ETL** : RudderStack’s [Reverse ETL](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/>) feature lets you send the enriched warehouse data and unified customer profiles to any downstream tool for activation use cases like personalization, engagement, and more. See the [Reverse ETL Quickstart](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/quickstart/>) guide to set up and activate your Reverse ETL pipeline.


## Features quickstart

  * RudderStack’s [Transformations](<https://www.rudderstack.com/docs/transformations/overview/>) feature lets you write custom functions in JavaScript or Python to implement specific use cases on your event data.

  * You can leverage [Tracking Plans](<https://www.rudderstack.com/docs/data-governance/tracking-plans/>) to enforce strict data quality checks on your inbound source events and standardize the events and properties sent to downstream tools using [Data Catalog](<https://www.rudderstack.com/docs/data-governance/data-catalog/>) and real-time schema fixes.

  * RudderStack [Profiles](<https://www.rudderstack.com/docs/profiles/overview/>) lets you leverage the real-time events and user traits collected in your warehouse to establish a comprehensive identity graph to create a customer 360 view.

  * With the [RudderStack Data Apps](<https://www.rudderstack.com/docs/profiles/data-apps/>), you can leverage the power of customer 360 to accelerate key projects like:

    * [Attribution](<https://www.rudderstack.com/docs/profiles/data-apps/attribution/>): Deliver report-ready attribution data for paid campaigns with minimal configuration and full transparency.
    * [Propensity Scores](<https://www.rudderstack.com/docs/profiles/data-apps/propensity/>): Run predictive models to forecast user churn or conversion, and sync these scores automatically to user profiles in business tools.
    * [Real-time Personalization](<https://www.rudderstack.com/docs/profiles/data-apps/real-time-personalization/>): Drive real-time website and app personalization, enhancing user engagement and conversion rates.
