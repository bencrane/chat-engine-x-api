# Get Started

Create a Profiles project in RudderStack.

* * *

  * __less than a minute

  * 


> ![info](/docs/images/info.svg)
> 
> See [Glossary](<https://www.rudderstack.com/docs/archive/profiles/0.10/glossary/>) to understand the key terms used in this documentation.

There are two ways to create a Profiles project in RudderStack:

  * Using the Profiles UI
  * Using the Profile Builder CLI


## Set up profiles in the RudderStack dashboard

Create a new Profiles project in the [RudderStack dashboard](<https://app.rudderstack.com>).

To get started, see the [Profiles Quickstart guide](<https://www.rudderstack.com/docs/archive/profiles/0.10/get-started/quickstart-ui/>).

## Use the Profile Builder CLI

**Profile Builder (PB)** is a command-line interface (CLI) tool that simplifies the transformation of data within your warehouse and generates customer profiles by stitching data together from multiple sources.

To get started see our [Profile Builder CLI](<https://www.rudderstack.com/docs/archive/profiles/0.10/get-started/profile-builder/>) documentation.

> ![info](/docs/images/info.svg)
> 
> Use this CLI to enhance a profiles project created within the **Profiles UI**.
