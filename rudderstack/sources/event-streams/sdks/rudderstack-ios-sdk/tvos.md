# tvOS

Use the RudderStack iOS (Obj-C) SDK for tvOS.

* * *

  * __less than a minute

  * 


[tvOS](<https://developer.apple.com/tvos/>) is Apple’s operating system for Apple TVs that are second generation or later. tvOS is based on the iOS operating system.

The tvOS platform also offers a tvOS SDK that you can leverage to develop apps that deliver engaging and interactive user experiences coupled with stunning picture quality and immersive sounds.

## iOS (Obj-C) SDK support for tvOS

Starting from version **1.1.0** , the [RudderStack iOS (Obj-C) SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/>) supports the tvOS platform.

> ![success](/docs/images/tick.svg)
> 
> You can now integrate the iOS (Obj-C) SDK with your tvOS apps and seamlessly track user events without the need for **any** additional configuration.

## Supported events

The tvOS platform supports all events that are supported by the [iOS (Obj-C) SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/>). These include:

  * [Track](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#track>)
  * [Screen](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#screen>)
  * [Identify](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#identify>)
  * [Group](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#group>)
  * [Alias](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#alias>)
  * [Reset](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#reset>)
