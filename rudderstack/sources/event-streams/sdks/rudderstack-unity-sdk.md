# Unity SDK

Use the RudderStack Unity SDK to send event data from your games to various destinations.

* * *

  * __less than a minute

  * 


The **RudderStack Unity SDK** is a wrapper for the RudderStack [Android (Java) SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-android-sdk/>) and [iOS (Obj-C) SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/>). It is used for tracking game event data.

After integrating this SDK with your game, you will be able to track your game event data and send it to your specified destinations via RudderStack.

Refer to the respective documentation depending on your preferred SDK version:

  * [SDK v1](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-unity-sdk/v1/>)
  * [SDK v2](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-unity-sdk/v2/>)
