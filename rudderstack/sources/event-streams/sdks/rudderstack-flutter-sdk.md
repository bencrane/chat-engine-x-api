# Flutter

Use the RudderStack Flutter SDK to send events from your Flutter applications to various destinations.

* * *

  * __less than a minute

  * 


RudderStack’s Flutter SDK lets you track and send the events from your Flutter apps to the specified destinations.

[![Github Badge](https://img.shields.io/pub/v/rudder_sdk_flutter)](<https://pub.dev/packages/rudder_sdk_flutter>)

Find the open source code for the [Flutter SDK](<https://github.com/rudderlabs/rudder-sdk-flutter>) on GitHub.

> ![info](/docs/images/info.svg)
> 
> See the [Flutter SDK v1 documentation](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-flutter-sdk/flutter-v1/>) if you are using an older version of the Flutter SDK.

[Installing the SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-flutter-sdk/installation-and-setup/>)| Install and set up the Flutter SDK.  
[Supported API](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-flutter-sdk/supported-api/>)| Supported API calls to track the event data.
