# Setup Guide Beta

Set up and configure Lytics Bulk Upload as a destination in RudderStack.

* * *

  * __less than a minute

  * 


This guide will help you set up Lytics Bulk Upload as a destination in RudderStack when connecting to a [Reverse ETL source](<https://www.rudderstack.com/docs/destinations/reverse-etl-destinations/lytics-bulk-upload/connect-retl-source/>).

> ![info](/docs/images/info.svg)
> 
> This guide assumes you have already set up a [Reverse ETL source](<https://www.rudderstack.com/docs/sources/reverse-etl/>) in RudderStack.

## Setup

In your [RudderStack dashboard](<https://app.rudderstack.com/>), add a source. Then, from the list of destinations, select **Lytics Bulk Upload**.

### Connection settings

Setting| Description  
---|---  
Name| Assign a name to uniquely identify your destination in RudderStack.  
Lytics Account ID| Enter your Lytics [account ID](<https://support.lytics.com/hc/en-us/articles/115001231351-How-do-I-find-my-Lytics-account-ID>).  
Lytics API Key| Enter your Lytics [API key](<https://support.lytics.com/hc/en-us/articles/115002118171-How-do-I-find-my-API-Key>).
