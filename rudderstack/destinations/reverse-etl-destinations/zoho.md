# Zoho Reverse ETL Destination Beta

Send your event data from RudderStack to Zoho.

* * *

  * __less than a minute

  * 


[Zoho](<https://www.zoho.com/>) is a cloud software suite for businesses. It provides features like a comprehensive CRM for customer-facing teams, accounting tools, HR and payroll management, and more.

RudderStack supports sending data to Zoho from your [Reverse ETL source](<https://www.rudderstack.com/docs/sources/reverse-etl/>).
