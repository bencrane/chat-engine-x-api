# Attentive Tag Destination

Send your event data from RudderStack to Attentive.

* * *

  * __less than a minute

  * 


[Attentive](<https://www.attentivemobile.com/>) is a text message marketing platform. It provides personalized text messaging solutions to help you create customer-first connections. It also provides an [Attentive Tag](<https://help.attentivemobile.com/hc/en-us/articles/10905920909460-The-Attentive-tag>) which you can add to your website to capture events and behavioral data, track revenue, and more.
