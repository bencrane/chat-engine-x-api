# Dub Destination Beta

Send your event data from RudderStack to Dub.

* * *

  * __less than a minute

  * 


[Dub](<https://dub.co/>) is a link attribution platform that helps you track the performance of your links and campaigns. It provides analytics, attribution tracking, and conversion optimization tools to measure the effectiveness of your marketing campaigns and identify which channels drive the most valuable conversions.

## Key features

The Dub destination integration supports the following features:

Feature| Available| Description  
---|---|---  
Connection modes| Cloud| Dub supports only cloud mode (Event Stream) for event delivery  
Message types| Yes| Supports `track` events for conversion attribution  
Event attribution| Yes| Tracks conversions back to specific campaign clicks  
Custom event mapping| Yes| Map RudderStack events to Dub’s lead/sales conversion events  
Revenue tracking| Yes| Automatically handles amount conversion to cents  
Consent management| Yes| Integrates with OneTrust, Ketch, iubenda, and custom consent providers  
Batching support| No| Dub does not support event batching  
User deletion| No| Dub does not support user deletion
