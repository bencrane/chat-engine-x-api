# Iterable Destination

Send your event data from RudderStack to Iterable.

* * *

  * __less than a minute

  * 


[Iterable](<https://iterable.com/>) is a popular growth marketing platform that helps you maximize your customers’ interactions and lifetime value (LTV).

RudderStack supports Iterable as a destination where you can seamlessly send your event data.

This documentation consists of the following sections:

  * [Setting up Iterable in RudderStack](<https://www.rudderstack.com/docs/destinations/streaming-destinations/iterable/setting-up-iterable/>)
  * [Sending events via cloud mode](<https://www.rudderstack.com/docs/destinations/streaming-destinations/iterable/iterable-cloud-mode/>)
  * [Sending events via device mode](<https://www.rudderstack.com/docs/destinations/streaming-destinations/iterable/iterable-device-mode/>)
