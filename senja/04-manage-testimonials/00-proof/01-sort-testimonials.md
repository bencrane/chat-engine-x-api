# Can I sort testimonials in my dashboard?

Testimonials in your dashboard are automatically sorted by date, with the newest testimonials appearing first.

It's not possible to sort by name, platform, rating, or change the timestamp sorting from oldest to newest.

## The workaround

You can use filters to find specific testimonials based on criteria like source, tags, rating, or testimonial type.

To learn how to filter testimonials, see How do I search or filter testimonials.
