# How do I edit a testimonial?

After collecting testimonials, you can edit their content, ratings, images, and metadata to ensure they display exactly how you want them across your website and marketing materials.

## How to edit testimonial content

1. Go to your Proof page: https://app.senja.io/
2. Select the testimonial you want to edit
3. You will be directed to the customer's page
4. Click Edit
5. Edit the content and click Save when you're done

## Things you can do

- Add translation to multiple languages
- Add or update Testimonial Title
- Add or update Rating
- Edit the Testimonial text
- Highlight the Testimonial text
- Add or update Images
- Change the Language info
- Add or update the Testimonial Source
- Add or update the Source URL
- Update the Date of submitting the testimonial

> For now, editing testimonial text is only doable on the responses of primary question.
