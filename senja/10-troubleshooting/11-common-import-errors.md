# Troubleshooting Common Import Errors

When importing testimonials into Senja, you might occasionally encounter an error.

This guide covers the most common issues and how to resolve them."Something went wrong" ErrorThis is a generic error message that can occur for several reasons.

If you encounter this error, please try the following troubleshooting steps:Refresh the page and try again: Sometimes, temporary connectivity issues can cause this error.

Check the URL format: Ensure you are using the correct and complete URL for the platform you're importing from.

See the platform-specific guidance below.

Try a different import method: If direct import fails, try using the Senja Chrome Extension or manual import.

Product Hunt Import: When importing from Product Hunt, you must use the URL of a specific launch, not the general product page.

For example, use a URL like https://www.producthunt.com/products/your-product-name/launches/your-product-name instead of https://www.producthunt.com/products/your-product-name.

Platform-Specific URL and Import Issues For detailed instructions on a specific platform, please refer to our dedicated import guides:G2Google Reviews/BusinessShopifyAirBNBYelpZillowApple PodcastsFiverrLinkedInIncorrect URL: Make sure you are using your main profile URL (e.g., https://www.linkedin.com/in/your-profile/), not a link to your recommendations or a specific post.

AmazonRegional sites: We are expanding our support for regional Amazon sites.

If you are having trouble with a non-US Amazon URL (like .es or .co.uk), please let us know.

GoogleSync Failures: If your Google reviews are not syncing, try re-syncing your account from the import page.

If the issue persists, contact support.

Business Not Found: Ensure you're using the correct business name and address as it appears on Google Maps.

Product HuntIncorrect URL: Use the URL for a specific launch, not the general product page, to import comments.

The format is https://www.producthunt.com/posts/your-product-name.

FiverrIncorrect URL: Use your public profile URL in the format fiverr.com/username.

TrustpilotSync Failures: If new reviews aren't syncing, try a manual sync.

If that fails, please reach out to our support team.

File Upload and Processing IssuesVideo UploadsUpload Failed: If a video file fails to upload, check your internet connection and try again.

If the problem continues, the file might be in an unsupported format or too large.

Video is Upside Down: If your video appears upside down, please contact support and share the video file with us so we can investigate.

Account and Plan Limitations Your ability to import testimonials may be limited by your current plan.

Testimonial Limit Reached: If you are on the free plan, you may have reached your testimonial limit.

To import more, you will need to upgrade your account.

Feature Not Available: Some import features, like auto-syncing, are only available on paid plans.

General Tips for Smooth ImportingPatience with Large Imports: If you are importing a large number of testimonials, the process may take a few minutes.

Please leave the import window open until it completes.

Dealing with Duplicates: If you notice duplicate testimonials after an import, please let us know so we can assist.

Filtering: If you can't find your imported testimonials, check if you have any filters active on your testimonials page.

If you've tried these steps and are still having trouble, please don't hesitate to contact our support team.

Provide as much detail as possible, including the import source, any error messages, and the URL you are using.
