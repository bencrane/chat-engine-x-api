# How to Hide App Store Icons Using CSS

You can hide app store icons on your website using a simple CSS snippet.

## Hide app store icons

To hide the app store icons, apply the following CSS code to your website. This code sets the display property to none for the specific icon class.

```css
.sj-integration-fixed-icon {
  display: none !important;
}
```
