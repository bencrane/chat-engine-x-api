# Why won’t all of my testimonial videos download in Chrome?

Want to download multiple video testimonials smoothly in Chrome?

You’ll need to enable automatic downloads.

This allows sites (like Senja) to download multiple files at once.

Here's how!

Open Chrome SettingsClick the three-dot menu in top right corner of your Chrome windowSelect "Settings"Navigate to Site SettingsFind Automatic DownloadsAllow Multiple File DownloadsTroubleshootingRestart Chrome after changing settingsEnsure you're on latest Chrome version
