# Does Senja provide security documentation or complete security questionnaires?

Senja does not currently offer security documentation or complete security questionnaires for customers.

Why we don't offer this serviceOur current pricing structure doesn't make it feasible to provide custom security documentation or questionnaire completion services.

This allows us to keep our platform affordable and accessible for businesses of all sizes.

What this means for youIf your organization requires completed security questionnaires or custom security documentation as part of your vendor evaluation process, you'll need to handle this independently or consider alternative solutions that offer these services.
