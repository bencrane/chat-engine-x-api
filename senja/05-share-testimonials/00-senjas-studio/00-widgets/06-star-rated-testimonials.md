# How to select only star-rated testimonials for a widget

By default, the Rating Badge and Testimonial Avatars widgets count all testimonials, whether they have a star rating or not. To show only star-rated testimonials, create a collection with just those testimonials and filter your widget to use it. Before you begin: You'll need admin access to create collections and manage widgets.Creating widget with 1 rating scaleGo to your Studio page: https://app.senja.io/studioChoose the widget that you've created or create a new one with Rating Badge or Testimonial Avatars templateClick Select TestimonialsSet the Filters > RatingAt the moment, you can only choose 1 rating (4-stars only, 5-stars only, etc.)See the testimonial previewIf you choose Auto-add, all testimonials with the chosen rating will be automatically updated (including the new ones coming after this setup)If you choose Manual selection, all testimonials with the chosen rating will be shown in the preview, you can cherry-pick which ones you wantClick Save Filters or Save Selection Your widget will include only the testimonials with certain ratingThe count will be based on the number of testimonials you chose, not the whole testimonials count you have collectedClick Save changes on the widget preview pageCreating widget with multiple ratingsIf you want to include different ratings (1 to 5 stars) in your widget, here's how:Go to your Proof page: https://app.senja.io/Filter the testimonials using Rating parameterFilter per rating, following the guide How do I search or filter testimonials? Add tags in bulk when you get the filtered testimonials view, following the guide How to add tags to multiple testimonials at onceDo this with the next rating that you wantLet's say you tag it as Star-ratedGo to your Studio page: https://app.senja.io/studioChoose the widget that you've created or create a new one with Rating Badge or Testimonial Avatars templateClick Select TestimonialsSet the Filters > Tags, choose the tag Star-ratedIf you choose Auto-add, all testimonials with the chosen tag will be automatically updated (including the new ones coming after this setup)If you choose Manual selection, all testimonials with the chosen tag will be shown in the preview, you can cherry-pick which ones you wantClick Save Filters (for Auto-add mode) or Save Selections (for Manual selection) All testimonials with star ratings (that have been tagged) will be added to the widgetClick Save changes on the widget preview page
