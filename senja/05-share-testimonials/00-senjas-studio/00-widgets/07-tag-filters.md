# Using tag filters in studio widgets

Tag filters let you control which testimonials appear in your widget. You can use OR logic to show testimonials with any selected tag, or AND logic to show only testimonials that have all selected tags. Tag filters only work in Auto-add mode. Manual selection disables automatic updates.Using OR and AND tag filtersTo set up tag filters for your widget:Open your widget in Studio and go to the Design tabClick Select Testimonials in the left sidebar Choose Auto-add mode and click Continue Expand the Tags section and select the tags you want to filter byUse the toggle to choose your filter logic:OR (default): Shows testimonials with any of the selected tagsAND: Shows only testimonials with all selected tags  Use OR when you want broad coverage (e.g., "Product" OR "Service" shows all testimonials tagged with either). Use AND when you need precise matches (e.g., "5-star" AND "Video" shows only video testimonials that are also tagged 5-star).How manual selection affects auto-addIf you choose Manual selection instead of Auto-add, you can still use tag filters to find testimonials, but your widget will not automatically update when new testimonials arrive. Once you manually select specific testimonials, your widget switches to a fixed list. To re-enable automatic updates, click Select Testimonials again and choose Auto-add mode.
