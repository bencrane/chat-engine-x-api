# How to choose testimonials with specific language in my widget?

If you want to create a widget to have the testimonials with a specific language, Senja provides the feature for you!Choosing testimonials with specific languageGo to your Studio page: https://app.senja.io/studioOpen your widgetClick Select TestimonialsClick Filter > Language Choose the language that you want from the dropdownClick Save Selection
