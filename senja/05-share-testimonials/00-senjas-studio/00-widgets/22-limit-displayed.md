# How to limit the number of testimonials displayed in a widget

You can control how many testimonials appear in your widget using the max testimonials setting.Widgets with testimonial limitation featureThis function is available on some widgets, not all:Image GalleryHero QuotesHighlights BoldBricksLindoy ListTestimonial MasonryBubble ListLimit testimonials in your widgetGo to your Studio page: https://app.senja.io/studioOpen your widgetGo to Design tabScroll down until you find Limit Testimonials section, put the toggle on Update the Max TestimonialsUpdate the Load more button label (default: Load more)Put the toggle on to Hide load more button (don't do it if you limit the number of testimonials)Click Save changes when you're done  The widget will randomly select testimonials up to your maximum limit each time it loads.
