# How to choose testimonials from multipart responses for the widget

When you collect testimonials using multi-question forms, each submission creates a multipart response with multiple answers. You can control which responses appear in your widget. This feature is available for testimonials collected through Forms 2.0 with multiple questions.

## Choosing testimonials from multipart responses

1. Go to your Studio page: https://app.senja.io/studio
2. Open your widget (in Saved tab) or create a new widget
3. Click Select Testimonials tab
4. Make sure it's on Manual selection (if it's on Auto-add, click Change how testimonials are added to switch to Manual selection)
5. Click Multipart on the Filters sidebar
6. Choose Show all responses from the dropdown
7. The multipart responses are now displayed, not only the primary
8. Tick the checkmark to select the testimonials (you can choose only the 2nd, 3rd, or 4th response, or only the primary and not choosing the others)
9. Click Save Selection when you're done

## How the multipart filter works

Select one of three options:

- **Show primary response only** - Displays only the answer of primary question from each testimonial (default)
- **Show all responses** - Expands each multipart testimonial into separate selectable responses
- **Use Multiclip (video responses only)** - Combines video responses only; resets your current selection

In the widget, multipart responses will be displayed as separate testimonials. For example, you have form with 3 questions, and your customers submit your testimonials through this form -- the widget will show 3 boxes of different testimonials from the same customer.

## How does it work in Auto-add mode

When you Select Testimonials in Auto-add mode:

- Multipart filter doesn't impact the testimonials to select
- Multipart filter just helps you to see the list of multipart responses
- It still makes sense to use the filter for Use Multiclip (video responses only) to only include the multipart responses that are all in video form

> Use the Forms filter to narrow down to specific questions for even more control over which responses appear.
