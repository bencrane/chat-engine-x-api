# How do I add or create a new Project?

Projects let you organize testimonials by client, brand, or product line.

Free and Starter plans include 1 project, while Pro includes 5 projects with the option to add more at $10/month each.

You need a Pro plan ($59/month) to create more than 1 project.

Only the project owner needs Pro—team members don't need their own subscription.

Add a new project
1.

Click on the project switcherOpen the project switcher from the top navigation bar. 
2.

Click on Create a New ProjectIn the dropdown menu, select Create a New Project. 
3.

Enter the project details and click Create ProjectFill in your project name and website URL, then click Create Project.

Purchase additional project slotsIf you've reached your 5-project limit on the Pro plan, you can purchase additional slots at $10/month per project.1.

Click on the project switcherOpen the project switcher from the top navigation bar. 
2.

Click on the Buy more projects buttonWhen you're at your project limit, you'll see a Buy more projects button. 
3.

Select how many projects to addChoose the number of additional projects you want and click Buy more projects.
