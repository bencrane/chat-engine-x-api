# Import testimonials from web page

Import testimonials from any web page into Senja. Before you begin: Have the public URL of the web page you want to import from.

**Account type:** All plans can run one-off imports from a web page.

## Option 1: Import from web

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Web Page icon
3. Paste the link to the Website Page field
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your web page testimonials are now in Senja and ready to use.

## Other methods

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose Web Page.

### Senja Chrome extension

You can use Senja Chrome extension following this guide: Import testimonials with the Chrome Extension.
