# Import testimonials from Google Play Store

Senja lets you import reviews from your Google Play Store app listing to use as social proof. This feature is available on all plans. Auto-sync is not currently available for Google Play Store.

## What you can do

- Import all reviews from your Play Store page at once
- Select individual reviews to import specific testimonials

## How to import from Google Play Store

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Play Store icon
3. Paste your link under Google Play Store app URL
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your Google Play reviews are now in Senja and ready to display.

## Other methods

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose Play Store.

### Senja Chrome extension

You can use Senja Chrome extension following this: Import testimonials with the Chrome Extension.

Need help? If you run into any issues importing your Google Play Store reviews, contact Senja Support and we'll help you get set up.
