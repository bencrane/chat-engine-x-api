# Import testimonials from AppSumo

Import your AppSumo reviews into Senja so you can share them as social proof. Before you begin: you'll need the public AppSumo product URL (for example, https://appsumo.com/products/your-product).

## What you can do

- Import all reviews from your AppSumo product page at once
- Select specific reviews to import only the testimonials you want
- Set up auto-imports for new AppSumo reviews from the auto-import page

## How to import from AppSumo

1. Go to Import from web
2. Click the AppSumo icon
3. Paste your link under AppSumo product URL
4. Click Import testimonials
5. Select the testimonials you want to import, or choose all to import everything
6. Click Import testimonials to finish

If you see a duplicates prompt, choose whether to allow duplicates or skip them before finishing the import.

Your AppSumo reviews are now in Senja and ready to share.
