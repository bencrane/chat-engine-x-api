# Import testimonials from Apple's App Store

Senja lets you import reviews from your Apple App Store listing to use as social proof. This feature is available on all plans. Auto-sync is not currently available for App Store.

## What you can do

- Import all reviews from your App Store page at once
- Select individual reviews to import specific testimonials

## How to import from App Store

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the App Store icon
3. Paste your link under App Store app URL
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your App Store reviews are now in Senja and ready to display on your website.

## Other methods

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose App Store.

### Senja Chrome extension

You can use Senja Chrome extension following this: Import testimonials with the Chrome Extension.

Need help? If you run into any issues importing your App Store reviews, contact Senja Support and we'll help you get set up.
