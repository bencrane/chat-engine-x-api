# Import testimonials from TikTok

Import TikTok videos into Senja using Import from web. Before you begin: Have the public URL of the TikTok video you want to import.

**Account type:** All plans can run one-off imports from TikTok.

## How to import from TikTok

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the TikTok icon
3. Paste your link in TikTok video URL field
4. Click Import testimonials

Your TikTok testimonials are now in Senja and ready to use.

## Other methods

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose TikTok.

### Senja Chrome extension

You can use Senja Chrome extension following this guide: Import testimonials with the Chrome Extension.
