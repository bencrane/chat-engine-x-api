# Online Presence

## Description
Indicates whether a website is an e-commerce website selling products directly online.

## Tier
Core

## Values
- **"Yes"** - Website has online sales capability
- **null** - Insufficient data to determine

## Fields

| Field | Type | Description |
|-------|------|-------------|
| hasOnlineSales | String | Indicates online sales presence ("Yes" or null) |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
