# Name

## Description
The name of the operating location.

## Tier
Core

## Applies To
- Operating Location

## Key Details
Operating location names often differ from brand names and convey location-specific information.

**Example:** "Target - Crossgates Mall" combines both the brand identifier and distinctive location details.

## Data Fields

| Field | Type | Description |
|-------|------|-------------|
| name | String | The operating location name |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
