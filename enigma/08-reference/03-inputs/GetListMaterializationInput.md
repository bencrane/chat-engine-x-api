# GetListMaterializationInput

## Overview

An input type used to retrieve information about a list materialization by its unique identifier.

## Fields

### id
- **Type:** `ID!` (non-null scalar)
- **Description:** The unique identifier for the list materialization to retrieve

## Usage

This input type is utilized by the following GraphQL operations:

- `listMaterialization` query
- `getListMaterialization` query (deprecated)
