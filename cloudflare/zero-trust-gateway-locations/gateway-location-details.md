# Get Zero Trust Gateway location details

`GET /accounts/{account_id}/gateway/locations/{location_id}`

Get a single Zero Trust Gateway location.

## Parameters

- **location_id** (string, required) [path]: 
- **account_id** (string, required) [path]:
