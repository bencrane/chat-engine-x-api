# Delete Stream

`DELETE /accounts/{account_id}/pipelines/v1/streams/{stream_id}`

Delete Stream in Account.

## Parameters

- **force** (string, optional) [query]: 
- **account_id** (string, required) [path]: 
- **stream_id** (string, required) [path]: 

## Response

### 200

Indicates a successfully deleted Stream.

### 4XX

Indicates an error in listing Streams.
