# BlitzAPI - Values List - Company Types

> **Case-sensitive.** Incorrect values silently return zero results.

This page contains all the possible values for the `type` filter. These represent company/organization classification types from LinkedIn.

**Used in:**
- `POST /v2/search/companies` — `company.type.include` / `company.type.exclude` parameters
- Response field `type` in company search and company enrichment results
