# BlitzAPI - Values List - Continents

> **Case-sensitive.** Incorrect values silently return zero results. Note: these use Title Case with spaces.

This page contains all the possible values for the `continent` filter.

**Used in:**
- `POST /v2/search/companies` — `company.hq.continent` parameter
- `POST /v2/search/employee-finder` — `continent` parameter
- Response field `location.continent` in person results
- Response field `hq.continent` in company results
