# BlitzAPI - Values List - Job Levels

> **Case-sensitive.** Incorrect values silently return zero results.

This page contains all the possible values for the `job_level` filter. These represent seniority categories for person searches.

**Used in:**
- `POST /v2/search/employee-finder` — `job_level` parameter

| VP |
| Director |
| Manager |
| Staff |
