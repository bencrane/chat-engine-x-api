# BYOC Trunking API

Bring Your Own Carrier (BYOC) Trunking for Voice is a cloud based solution that allows you to bring your own voice carrier to Twilio, enabling your calls to use our Programmable Voice tools and APIs.

## Explore Twilio's BYOC resources:

- **ByocTrunk Resource**
- **ConnectionPolicy Resource**
- **ConnectionPolicyTarget Resource**
- **SIP Credential Resource**
- **SIP CredentialList Resource**
- **SIP CredentialListMapping Resource**
- **SIP Domain Resource**
- **SIP IpAccessControlList Resource**
- **SIP IpAccessControlListMapping Resource**
- **SIP IpAddress Resource** (for IpAccessControlList)
- **SIP IpRecord Resource** (for SourceIpMapping)
- **SourceIpMapping Resource**
