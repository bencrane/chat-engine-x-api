# VoiceDrop.ai - Phone Numbers - Other - List Sender Numbers

## GET - List Sender Numbers

```
GET https://api.voicedrop.ai/v1/sender-numbers
```

---

## Authorization

**API Key** - uses API Key from collection: VoiceDrop API Reference

```bash
curl --location 'https://api.voicedrop.ai/v1/sender-numbers'
```
