# Introduction

Sendoso implements SCIM Version 2.0 protocol so you can easily manage users in the Sendoso platform.

## Use Cases

- **Automated user provisioning** - Create users programmatically
- **Team assignment** - Assign users to teams
- **Role management** - Set and update user roles
- **User deprovisioning** - Deactivate users when they leave

## Getting Started

Users must first obtain a Sendoso account with client credentials before accessing the SCIM API.
