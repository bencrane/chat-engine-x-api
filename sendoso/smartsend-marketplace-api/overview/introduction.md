# Introduction

Sendoso offers a Marketplace and SmartSend API for automating physical gift and direct mail delivery.

## Key Features

- **Marketplace** - Send from a large selection of gifts and direct mail options
- **SmartSend** - An intelligent selection system that determines optimal gifts based on recipient preferences
- **Automation** - Automate sending to customers, leads, and employees

## Getting Started

Users need a Sendoso account and API key to begin.

## Technical Details

- REST-based API using JSON format
- Standard HTTP response codes
- OAuth 2.0 authentication
