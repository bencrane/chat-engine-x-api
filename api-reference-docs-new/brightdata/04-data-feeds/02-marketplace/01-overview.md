# Dataset Marketplace Overview

> The Dataset Marketplace is a centralized platform providing regularly updated, ethically sourced datasets from multiple domains, offering flexible solutions for data customization and acquisition.

## Introduction to the Dataset Marketplace

The Dataset Marketplace is a one-stop platform for discovering, customizing, and purchasing high-quality datasets from over 120 domains. Datasets are regularly updated and ethically sourced from publicly available information, ensuring both accuracy and compliance. Whether you need ready-to-use or on-demand solutions—covering fields like social media, real estate, B2B data, or AI training—flexible purchase options and comprehensive customization make it easy to get the data you need.

## Get Started

- **Explore** — Browse a wide range of pre-built, validated datasets across multiple domains.
- **Dataset view** — Examine data samples, data-fields, and customization options before purchase.
- **Filter Dataset** — Refine datasets with advanced filters, hidden fields, and custom subsets.
- **Download data sample** — Preview and download a sample of any dataset in multiple formats.
- **Fill rates and statistics** — Review data completeness and field coverage to ensure accuracy.
- **Purchase options and pricing** — Understand one-time purchases, subscriptions, and other flexible pricing models.
- **Data delivery and export** — Learn how to receive datasets via email, API, webhooks, and various cloud services.
