# Financial Spending

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/v2/financial_spending/major_object_class/` | GET | Returns financial spending data by major object class for the latest quarter based on the given fiscal year |
| `/api/v2/financial_spending/object_class/` | GET | Returns financial spending data by object class for the latest quarter based on the given fiscal year |
