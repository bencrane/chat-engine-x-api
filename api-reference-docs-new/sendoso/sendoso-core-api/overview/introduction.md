# Introduction

The Sendoso API enables automation of physical gifting and direct mail through REST-based integration.

## Key Capabilities

- Programmatically trigger sends (e.g., re-activate stale leads in your pipeline)
- Embed sending functionality into custom applications
- Conduct targeted marketing outreach based on specific criteria

## Getting Started

Users must establish a Sendoso account and retrieve an API key before making requests.

## Technical Details

- Architecture follows REST principles with resource-oriented URL structures
- HTTP status codes indicate errors
- Supports GET and POST request methods
- Data exchange uses JSON format exclusively
