# Voiceflow Integration with Vapi

## Overview

Voiceflow functions as a conversational AI platform enabling teams to "build, manage, and deploy AI agents, especially chatbots, to enhance customer experiences." The platform features a drag-and-drop interface that allows non-technical users to create sophisticated chatbot interactions for customer support automation.

To integrate Vapi with Voiceflow, you must establish a proxy server. This intermediary processes requests originating from Voiceflow by forwarding them to Vapi's text completion API and returning the corresponding responses. The proxy requires hosting on your own infrastructure to facilitate communication between the two platforms.
