# Sympana Connector Documentation

## Overview
Sympana Connector integrates Vapi agents with GoHighLevel, eliminating the need for middleware tools like Make or Zapier. The solution manages outbound sequences, inbound routing, data syncing, and CRM logic directly within GHL.

**Compliance Note:** The service meets HIPAA and GDPR standards for regulated industries including healthcare and finance.

## Core Features

- **Outbound Calling**: Trigger Vapi calls from GHL workflows and CRM events
- **Inbound Agent Support**: Deploy receptionist agents with routing and voicemail capture
- **Native GHL Automation**: Manage calendars, contacts, tags, and opportunities without API mapping
- **Timezone Intelligence**: Automatically reach leads during optimal contact windows
- **No Backend Required**: Pre-built for Vapi + GHL integration
- **Enterprise-Grade**: Includes retry logic and observability for high-volume operations

## Implementation Steps

1. **Install connector** in your GoHighLevel agency
2. **Connect Vapi agents** via Sympana's setup form
3. **Select automation flows** from 50+ prebuilt templates
4. **Deploy workflows** directly within GHL

## Differentiation from Native Integration

Sympana extends Vapi's native GHL connector by providing "workflow-level control" and "bidirectional call handling," managing full campaign sequences rather than isolated actions. It handles pipeline syncing, timezone logic, and operational scaling that the basic integration cannot.

## Pricing Model

Most actions are free, with costs only for **Place Call Trigger**, **Get Call**, and **Wait (For Lead's Time)**. Qualification flows, bookings, and CRM syncing operate at no additional expense.
