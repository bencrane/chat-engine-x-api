# AWS S3 Documentation Summary

## Overview
The page describes how to configure AWS S3 storage for recording chat conversations through the Vapi dashboard. Users must set up credentials and bucket details in the "Cloud Providers" section.

## Key Configuration Requirements

The documentation outlines four essential settings:

1. **AWS Access Key ID** - "The access key id for AWS"
2. **AWS Secret Access Key** - "The secret access key for AWS"
3. **S3 Bucket Name** - "The name of the bucket to upload recordings to"
4. **Bucket Path Prefix** - Supports optional path prefixes with LiquidJS date templating

## Notable Features

The path prefix feature allows dynamic directory structures using template syntax like `{{ "now" | date: "%Y/%m/%d" }}` to organize recordings by date hierarchies.
