# Google Transcriber Documentation

## Overview

The page explains that "Gemini is Google's latest artificial intelligence (AI) initiative, developed by Google DeepMind" with capabilities across multiple modalities.

## Setup Instructions

To configure Google as your transcriber in Vapi:

1. Navigate to the Assistants section within your Vapi dashboard
2. Select your assistant and access the Transcriber tab
3. Choose "google" from the Provider dropdown menu (you can also customize the model and language settings here)
4. Click Publish to activate the transcriber and begin conversations

## Language Support

The documentation notes that Gemini supports "a wide range of languages" by default through its multilingual capability, though users may select specific language preferences using the dropdown menu if desired.
