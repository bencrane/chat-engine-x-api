# Inworld Voice Provider Documentation

## What is Inworld?

Inworld develops AI products for application builders, featuring a text-to-speech service with "state-of-the-art voice AI radically more accessible for developers." The platform prioritizes low-latency streaming suitable for applications needing quick audio responses.

## Key Features

Inworld TTS leverages advanced LLM-based speech models to deliver natural, emotionally expressive voices. Primary capabilities include:

- **Real-Time Synthesis**: The system delivers "the first 2-second audio chunk in as few as 200ms," enabling responsive conversational interactions.
- **Multilingual Capability**: Supports 11 languages including English, Spanish, French, Korean, and Chinese for global audience reach.
- **Developer API**: Comprehensive API documentation with real-time streaming support and customizable voice parameters.

## Applications

Developers can implement Inworld TTS across:

- **Interactive Services**: Customer service bots, virtual assistants, and interactive characters with natural speech
- **Media Production**: Professional voiceovers for videos and podcasts
- **Educational Platforms**: Expressive narration for e-learning content

## Vapi Integration

Inworld voices integrate directly into Vapi. Users access Voice Configuration in the Vapi dashboard, select Inworld as the provider, choose their preferred language and voice, then publish to deploy.
