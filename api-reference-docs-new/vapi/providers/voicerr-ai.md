# Voicerr AI - Whitelabel AI Voice Agents

## Summary

Voicerr AI is a white-label platform powered by Vapi that enables agencies to offer voice AI services under their own branding. The service provides "unlimited agents, clients & sites" along with automated billing, real-time call analytics, and batch-calling capabilities.

## Key Features

The platform distinguishes itself through several core offerings:

- **Instant Branding**: Agencies can launch a custom-branded portal with their logo and domain within minutes
- **Vapi Integration**: One-time API key connection syncs assistants, phone numbers, and call data automatically
- **Automated Billing**: Stripe integration handles invoicing and payment processing with customizable markups
- **Outbound Capabilities**: Batch calling tools enable high-volume prospecting campaigns
- **Growth Tools**: Built-in Lead Finder and Automation Lab for workflow automation

## Use Cases

The platform supports multiple business models: agencies can resell voice AI agents to clients, offer outbound calling services for lead generation, or package complete contact-center solutions under their own brand.

## Getting Started

Agencies connect by providing their Vapi Organization ID and Private Key, then customize pricing tiers and client permissions before onboarding customers.
