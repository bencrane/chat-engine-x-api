# Introduction to Tools

## Overview

The page describes how to extend an assistant's capabilities through function calling tools. According to the documentation, **"Tools allow your assistant to take actions beyond just conversation"** including call transfers, data access, and application triggers.

## Four Tool Types

The platform offers these tool categories:

1. **Default Tools** - Vapi-provided built-in functions for standard operations
2. **Custom Tools** - User-created webhook-based functions for system integration
3. **Code Tools** - TypeScript code executing directly on Vapi infrastructure
4. **Integration Tools** - Pre-built connectors to Make and GoHighLevel platforms

## Integration Tool Capabilities

The Make and GHL integration features enable users to import workflows and trigger automated processes via voice commands. Common applications include appointment booking, CRM updates, email/SMS notifications, order processing, and support ticket management.

## Key Technical Features

The documentation highlights four main capabilities:
- Function calling triggered by conversation context
- Synchronous and asynchronous operation modes
- Backend connectivity via webhooks
- Integrated error handling with fallback options
