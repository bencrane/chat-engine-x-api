# Outbound Campaigns Quickstart

## Overview

This guide helps you establish a personalized outbound campaign using Vapi for conducting post-service feedback and follow-up calls. The platform enables organizations to "improve customer experience and gather valuable insights" through automated voice interactions.

## Key Learning Objectives

The quickstart covers four main areas:
- Initial campaign setup incorporating customer data
- Customized feedback collection call configuration
- Campaign launch and performance monitoring
- Call outcome reports and performance metrics

## Requirements

Before starting, ensure you have:
- An active Vapi account
- A phone number provisioned through a provider like Twilio (Vapi's complimentary numbers aren't compatible)
- Customer data prepared as a CSV file
- A pre-configured Assistant

## Implementation Steps

### Campaign Launch Process

The setup involves seven steps:
1. Access the Vapi Dashboard
2. Navigate to the Outbound Campaigns section
3. Create a new campaign with appropriate naming
4. Configure your phone number source
5. Upload and validate your customer recipient list
6. Select your Assistant
7. Review settings and initiate the campaign

### Monitoring Your Campaign

After launch, you can track performance through the campaign dashboard, review completion metrics and call statuses, and access individual call logs containing duration, outcomes, transcripts, and recordings.
