# Support Documentation

## Title and Subtitle
**Support** - We are open to all kinds of help inquiry, feedback and feature request.

## Our Support Options

Vapi provides three primary support channels:

1. **Email Support** - Contact support@vapi.ai directly for assistance with your projects
2. **Community Support** - Access the Discord server at discord.com/invite/pUFNcf2WmH for collaborative help and support tickets
3. **Enterprise Support** - Dedicated 24/7 assistance available through enterprise plans

*Note: Enterprise customers receive fastest response times through dedicated support channels.*

## Feature Requests and Bug Reports

Users can contribute to Vapi's development by:
- Submitting feature requests on the public roadmap
- Reporting bugs encountered during platform usage
- Voting on community-requested features

*The platform actively monitors and prioritizes requests based on user engagement.*
