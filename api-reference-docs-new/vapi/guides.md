# Guides

This documentation page showcases real-world, cloneable examples for building voice agents using Vapi's Assistants and Squads frameworks.

## Available Examples

The page features 12 guides organized in a card-based layout:

**Assistant-Based Examples:**
- **Appointment Scheduling** - A barbershop scheduling assistant
- **Lead Qualification** - An outbound sales agent with automatic scheduling
- **Dynamic Multilingual Agent** - Automatic language detection with real-time switching
- **Support Escalation** - Intelligent routing based on customer tier and issue complexity
- **Docs Agent** - Documentation question-answering capability
- **Inbound Support** - Technical support with call history retention
- **Voice Widget** - Website integration tool for user interaction

**Squad-Based Examples:**
- **Medical Triage & Scheduling** - Patient triage and clinic appointment management
- **Ecommerce Order Management** - Order tracking and return processing
- **Property Management** - Dynamic call routing for tenant inquiries
- **Multilingual Support (Structured)** - Structured workflow with language selection

**Developer Tools:**
- **Vapi CLI** - Command-line tool for faster agent development, local testing, and IDE integration

Each guide includes a brief description and icon, with badges indicating whether it uses Assistants, Squads, or is a developer tool.
