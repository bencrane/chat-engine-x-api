# Name

## Description
The customer-facing version of the name that best represents the business.

## Tier
Core

## Methodology
Ranked by dataset quality and frequency from public and private sources.

## Special Handling
- **Persons**: Agents or providers associated with the business
- **Sub-brands/Franchises**: Specific location variations

## Fields

| Field | Type | Description |
|-------|------|-------------|
| name | String | The brand name |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
