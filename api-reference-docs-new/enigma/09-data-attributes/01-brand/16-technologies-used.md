# Technologies Used

## Description
Indicates third-party technologies used, primarily payment solutions.

## Tier
Premium

## Supported Technologies

### Website Technologies
- Adyen
- Braintree
- PayPal
- Shopify
- Stripe

### Operating Location Technologies
- Clover
- PayPal
- Shopify
- Square
- Stripe
- Toast

## Ranking System
- **Rank 0**: Most recent observation
- **Higher ranks**: Historical observations

## Fields

| Field | Type | Description |
|-------|------|-------------|
| technology | String | Name of the technology |
| category | String | Category of technology |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
