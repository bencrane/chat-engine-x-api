# Registered Entity

## Description
Businesses formally registered as legal entities with U.S. state Secretaries of State.

## Tier
Premium

## Data Source
State Secretary of State offices (ultimate source of truth).

## Consolidation
Unifies domestic and foreign registrations across states.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| name | String | Registered entity name |
| registeredEntityType | String | Type of registered entity |
| formationDate | String | Date of formation |
| formationYear | Integer | Year of formation |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
