# Website Content

## Description
State of the website at a particular time, crawled at least every 90 days.

## Tier
Plus

## Ranking System
- **Rank 0**: Most recent crawl
- **Higher ranks**: Historical crawls

## Fields

| Field | Type | Description |
|-------|------|-------------|
| faviconImage | String | Favicon image data |
| faviconUrl | String | URL to favicon |
| httpStatusCode | Integer | HTTP status code from crawl |
| websiteAvailability | String | Website availability status |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
