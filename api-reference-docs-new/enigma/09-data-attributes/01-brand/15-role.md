# Role

## Description
Roles which people and other legal entities hold at U.S. businesses.

## Tier
Plus

## Fields

| Field | Type | Description |
|-------|------|-------------|
| jobTitle | String | Job title of the role holder |
| jobFunction | String | Function of the job |
| managementLevel | String | Level of management |
| externalId | String | External identifier |
| externalUrl | String | External URL reference |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
