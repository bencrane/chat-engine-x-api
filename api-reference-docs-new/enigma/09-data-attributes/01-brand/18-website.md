# Website

## Description
A website associated with the business.

## Tier
Core

## URL Decomposition
The website attribute breaks down URLs into their component parts:
- Full website
- Domain
- Subdomain
- Top-level domain
- Path
- Fragment

## Fields

| Field | Type | Description |
|-------|------|-------------|
| website | String | Full website URL |
| domain | String | Domain name |
| subdomain | String | Subdomain portion |
| topLevelDomain | String | Top-level domain (.com, .org, etc.) |
| path | String | URL path |
| fragment | String | URL fragment |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
