# Brand Revenue Quality

## Description
Warnings and issues related to brand revenue, flagging financial anomalies.

## Tier
Plus

## Issue Types
- Revenue drops to zero
- 20% decrease from median
- 250% increase above median

## Severity Levels
- High
- Medium

## Fields

| Field | Type | Description |
|-------|------|-------------|
| issueDescription | String | Description of the revenue issue |
| issueReason | String | Reason for the flag |
| issueSeverity | String | Severity level (High/Medium) |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
