# Role

## Description
Roles which people (and other legal entities) hold at U.S. businesses.

## Tier
Plus

## Applies To
- Brand
- Operating Location
- Legal Entity

## Data Fields

| Field | Type | Description |
|-------|------|-------------|
| jobTitle | String | Observed job title from employee datasets |
| jobFunction | String | Standardized role category (e.g., "Accounting", "Contracts") |
| managementLevel | String | Hierarchical position within the organization |
| externalId | JSON | Reference identifier from external sources |
| externalUrl | String | URL linking to external role information |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
