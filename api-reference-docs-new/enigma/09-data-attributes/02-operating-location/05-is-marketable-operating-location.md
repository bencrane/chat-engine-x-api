# Is Marketable (Operating Location)

## Description
A boolean value indicating whether the operating location is marketable.

## Tier
Core

## Applies To
- Operating Location

## Marketability Criteria
- Active operational status with current information
- Revenue generated within the preceding 12 months
- Customer reviews posted within the past 12 months

## Data Fields

| Field | Type | Description |
|-------|------|-------------|
| isMarketable | Boolean | Core marketability indicator |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
