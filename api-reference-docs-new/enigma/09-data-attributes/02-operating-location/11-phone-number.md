# Phone Number

## Description
The phone number for a particular business entity.

## Tier
Core

## Applies To
- Brand
- Operating Location
- Legal Entity

## Format
A twelve-digit string representation of the complete phone number.

## Data Fields

| Field | Type | Description |
|-------|------|-------------|
| phoneNumber | String | Twelve-digit string representation |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
