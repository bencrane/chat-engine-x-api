# Website

## Description
A website associated with a business.

## Tier
Core

## Applies To
- Brand
- Operating Location

## Data Fields

| Field | Type | Description |
|-------|------|-------------|
| website | String | Complete, fully-formed URL with all components |
| domain | String | Domain portion (e.g., "enigma" from documentation.enigma.com) |
| subdomain | String | Subdomain component (e.g., "documentation") |
| topLevelDomain | String | TLD extension (e.g., "com") |
| path | String | URL path segment following the domain |
| fragment | String | Anchor/hash portion after the path |
| id | UUID | Unique identifier |
| firstObservedDate | String | Date first observed |
| lastObservedDate | String | Date last observed |
