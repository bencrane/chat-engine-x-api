# updateListMaterialization Mutation

## Overview

The `updateListMaterialization` mutation allows you to modify an existing list materialization within the Enigma GraphQL API.

## Signature

```graphql
updateListMaterialization(input: UpdateListMaterializationInput!): UpdateListMaterialization
```

## Parameters

**input** (required)
- Type: `UpdateListMaterializationInput!`
- Description: The input object containing the parameters needed to update a list materialization

## Return Type

**UpdateListMaterialization** object
- Contains the result of the update operation

## Related Mutations

- **cancelListMaterialization** — Stop an in-progress materialization
- **createListMaterialization** — Initiate a new list materialization
- **updateList** — Modify list properties
