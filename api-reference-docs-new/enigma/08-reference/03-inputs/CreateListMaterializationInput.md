# CreateListMaterializationInput

## Overview

The `CreateListMaterializationInput` is a GraphQL input type used to initiate the materialization of a list within the Enigma API.

## Fields

### `listId` (required)

- **Type:** `ID!`
- **Description:** The unique identifier of the list to materialize. This field is required.

## Usage

This input is utilized by the [`createListMaterialization`](/reference/graphql_api/mutations/create-list-materialization) mutation to trigger the materialization process for a specified list.
