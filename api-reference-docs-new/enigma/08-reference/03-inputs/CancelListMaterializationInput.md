# CancelListMaterializationInput

## Overview

This GraphQL input type is used to specify parameters for canceling a list materialization operation.

## Fields

### `id` (ID!)

A required, non-null identifier that specifies which list materialization should be canceled.

## Usage

The `CancelListMaterializationInput` type serves as the parameter object for the `cancelListMaterialization` mutation, enabling clients to halt an ongoing or pending list materialization process by providing its unique identifier.
