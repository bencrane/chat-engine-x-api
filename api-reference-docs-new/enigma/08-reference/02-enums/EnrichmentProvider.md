# EnrichmentProvider

## Overview

The `EnrichmentProvider` enumeration specifies which data enrichment service to use when enriching business information.

## Values

- **ZOOMINFO** - Enrichment data sourced from ZoomInfo
- **SEARCH** - Enrichment data sourced from Enigma's search capabilities

## Usage

This enum is used as part of the `EnrichmentInput` input type to specify which enrichment provider should be used when performing data enrichment operations via the GraphQL API.

## Related Types

- [`EnrichmentInput`](/reference/graphql_api/inputs/enrichment-input) - Input type that utilizes this enum
