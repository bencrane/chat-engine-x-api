# Departments

# Prospeo - Values List - Departments

There are 2 types of departments: normal departments, and headcount growth departments.

The normal departments are the ones found everywhere: person's department, used in the "Employee count by department" filter, etc. The headcount growth departments are a small subset of departments only used within the filter "Employee headcount growth by department". Except if using this filter, this can be ignored.
