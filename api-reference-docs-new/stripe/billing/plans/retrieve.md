### Parameters#### activebooleanOnly return plans that are active or inactive (e.g., pass false to list all inactive plans).
- #### productstringOnly return plans for the given product.
### More parametersExpand all- #### createdobject- #### ending_beforestring- #### limitinteger- #### starting_afterstring### ReturnsA dictionary with a data property that contains an array of up to limit plans, starting after plan starting_after. Each entry in the array is a separate plan object. If no more plans are available, the resulting array will be empty.
