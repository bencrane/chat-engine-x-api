# Can I Pre-Fill Collection Form Fields?

Yes, it's possible to pre-fill fields in your testimonial collection form.

The form fields you can pre-fill are:

- Name
- Email
- URL

Currently, it is not possible to pre-fill the 5-star rating field.

This helps your customers have an even simpler experience when leaving you a testimonial.

To do this, you need to append the following parameters to the form's URL. Like this:

```
?name=Oliver+Meakings&email=olly@senja.io&url=https://www.first100users.com
```

You need to update the URL to reflect your customers' details.

## An example form URL

Here's a Senja form URL with the parameters added:

```
https://www.senja.io/p/slogen/r/f2wPRV?name=Oliver+Meakings&email=olly@senja.io&url=https://www.first100users.com
```
