# Using Projects in Senja

Projects help you organize your testimonials, forms, widgets, team members, and Walls of Love into separate groups.

Use them to manage multiple businesses, separate client work, or organize different types of social proof.

Account accessFree: 1 project (cannot add more)Starter: 1 project (cannot add more)Pro: 5 projects (add more for $10/month per project)When to use projectsConsider using projects if you:Have multiple businesses, services, or offers (alternatively, you can use tags for this)Are an agency managing multiple clients (see agency plans)Want to separate different proof types (for example, one project for customer testimonials and another for employee testimonials)
