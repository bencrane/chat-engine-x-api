# How to move testimonials between different Senja accounts?

Transferring testimonials between separate Senja accounts requires help from our support team.

This is different from moving testimonials between projects in the same account.

This feature is available on all account typesHow to request a transferContact Senja Support team with:The email address for the source account (where testimonials currently are)The email address for the destination account (where you want them moved)Which testimonials you want transferred (or specify all)Our team will handle the transfer for you.

If you're moving testimonials within the same account between projects, you can do this yourself by duplicating them
