# How to check if an email address has a Senja account

You can check if an email address is already registered with Senja by using the login page.

This is helpful when you're unsure if you've already created an account or if you're managing multiple accounts.

Check for an existing accountGo to the Senja login page and enter the email address you want to check.

Click Continue.

Senja will show you one of two messages:Account exists — You'll receive a password reset or login link via emailNo account found — You'll see a message saying the email isn't registered If you're trying to log in but can't remember which email you used, try all email addresses you commonly use with this method.
