# How Do I Add a Rich Snippet?

You want to add a rich snippet to your search results. Senja makes it easy.

Improve your SEO and increase your click-through rate. Please note that rich snippets are only available on our Pro and Team plans. You can upgrade here.

## Adding a rich snippet

1. Go to Rich snippet page: https://app.senja.io/rich-snippet
2. Copy the code snippet
3. Paste the code snippet between the `<head></head>` tags on your website

Not sure what to do? Contact our Support.
