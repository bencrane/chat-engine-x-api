# Can I randomize the testimonials order in widget?

Yes, you can randomize testimonials so they shuffle every time your widget loads. This keeps your social proof fresh and shows different testimonials to different visitors.

## How to randomize testimonial order

1. Go to your Studio page: https://app.senja.io/studio
2. Go to the Design tab in your widget editor
3. In the Testimonials section, click Reorder testimonials
4. In the Sort testimonials dropdown, select Randomly
5. You can also pin some testimonials that you want to keep on the top rows
6. Click Save changes

> Testimonials will reshuffle every time the widget loads, so each visitor may see a different order.

## Other ordering options

If you don't want random order, you can also sort by:

- **Manually** - Drag and drop testimonials into your preferred order
- **By Date** - Newest or oldest first
- **By Rating** - Highest or lowest rated first
- **Videos first** - Show video testimonials before text ones
