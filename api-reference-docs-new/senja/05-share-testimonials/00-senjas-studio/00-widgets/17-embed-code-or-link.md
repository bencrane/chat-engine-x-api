# Where do I get my widget embed code or link?

You can get your widget embed code or link from two places: the Studio or the Chrome extension.From the StudioGo to your Studio page: https://app.senja.io/studioSee your saved assetsClick the 3-dots on the top-right corner of the widget you wantClick Copy Code or Copy URLCopy Code is for grabbing the whole code to embedCopy URL is for grabbing the link to view the widget Paste the code/link in the page you want to embed the widget toFrom the Chrome extensionWith Senja's free Chrome Extension you have instant access to all your proof, and your form, widget and Wall of Love links and embed code. Here's how to grab your widget embed code:Open the page where you want to embed the widget toOpen Senja Chrome extensionClick on Saved Click on Copy Link to or Copy Embed Paste the code/link in the page you want to embed the widget to
