# How to automatically populate widgets with high-rated reviews from multiple platforms

How to automatically populate widgets with high-rated reviews from multiple platformsYou can automatically import 4-5 star reviews from Trustpilot, Facebook, and Google, then filter them into separate widgets by platform.Set up auto import with star rating filtersNavigate to the Auto import featureSelect your review platforms (Trustpilot, Facebook, Google)Specify the star ratings you want to import (e.g., 4 and 5 stars only)Assign a unique tag to each source during setupThe imported testimonials will be automatically approved once the setup is complete.Create filtered widgets by platformGo to the StudioSelect or create your widgetIn the widget settings, filter testimonials by the tags you created during importRepeat this process to create separate widgets for each review sourceThis approach lets you showcase platform-specific reviews while maintaining quality control through star rating filters.
