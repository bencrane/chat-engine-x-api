# Video Download Troubleshooting: Complete Guide to Common Issues

This guide fixes the most common video download issues. For basic download steps, see How do I download my video testimonials?

## Chrome only downloads the first video

Chrome blocks multiple downloads by default. Allow them for Senja:

1. Open Chrome Settings (three-dot menu on the top right → Settings)
2. Go to Privacy and security → Site settings
3. Click Automatic downloads (usually hidden under Permissions → Additional permissions)
4. Set the toggle on for Sites can ask to automatically download multiple files or add app.senja.io to the Allowed to automatically download multiple files
5. Restart Chrome and try again

More details: Why won't all of my testimonial videos download in Chrome?

## Getting CSV files instead of MP4 videos

There are multiple scenarios on video download:

- Single question and the response is video: MP4 download from Proof page
- Multiple questions and all responses are video: MP4 download from Proof page
- Multiple questions and the primary response is video: MP4 download from Proof page
- Multiple questions, but the video is in non-primary question: only possible with CSV export

Follow the article How do I download my video testimonials? to see how it works in details.

If only some videos download, enable multiple downloads in Chrome (see above) and check your internet connection.

## Downloads not starting

- Check your browser allows downloads and has enough storage space
- Clear browser cache and cookies (Chrome Settings → Privacy and security → Delete browsing data)
- Disable ad blockers or try incognito mode

## Downloads fail or get corrupted

- Check your internet connection is stable
- Try a different browser (Firefox, Safari, Edge)
- Download one video at a time instead of bulk downloading

## Download button missing

- Refresh the page
- Confirm it's a video testimonial (not text)
- Check your account has download permissions

## Alternative methods

If standard downloads don't work:

- Use the Senja Chrome Extension — learn more
- Contact Senja Support for direct video URLs
- Right-click the video → Save video as in browser developer tools

Set up Chrome's multiple download settings before you need them, and download regularly rather than in large batches.

Still stuck? Contact Senja Support with your browser type, error messages, and which videos won't download.
