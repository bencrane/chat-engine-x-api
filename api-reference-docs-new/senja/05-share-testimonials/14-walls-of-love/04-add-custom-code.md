# Add Custom Code to My Wall of Love

You want to add a custom code to your Wall of Love. We make it easy.

For example, you can add your analytics code.

1. Click on Custom Domain on the menu
2. If you don't have one, add a custom domain (see How do I add a custom URL to my Wall of Love?)
3. Navigate to Custom Code
4. Paste your code snippet
5. Click Save
