# How to Set a Default Selected Tag on Your Testimonial Board

Setting a default selected tag on your testimonial board is currently not supported in Senja.

## Current behavior

When your testimonial board loads, it displays all testimonials by default. Users can then filter testimonials by selecting specific tags, but you cannot pre-select a tag to be active on page load.

## Request this feature

This functionality has been classified as a feature request. You can submit your idea to our feedback portal at https://feedback.senja.io.

When you submit your request:

- Other users can upvote your idea
- You'll receive notifications when the feature is implemented

## Workaround for video testimonials

If you're using video testimonials, you can create visual distinction by customizing video thumbnails:

- Use different thumbnails for each testimonial, even if they share the same video content
- This helps users differentiate between testimonials when viewing the full board

This approach makes it easier for visitors to identify specific testimonials without relying on tag filtering.
