# How do I edit my tags?

You can edit your tags to rename them, update descriptions, or change categories. When you edit a tag, the changes apply across all testimonials using that tag.

## Edit a tag

1. Go to your Tags page: https://app.senja.io/tags
2. Find the tag you want to edit
3. Click Edit next to the tag
4. In the Manage Tag modal, update any of these fields:
   - Change Category to organize your tags into groups
   - Tag name (required)
   - Tag description (optional)
5. Click Save

> Renaming a tag automatically updates it across all testimonials that use it.

## Delete a tag

If you no longer need a tag, you can delete it from the same modal:

1. Go to your Tags page: https://app.senja.io/tags
2. Click Edit next to the tag you want to delete
3. Click Delete to remove the tag

> Deleting a tag removes it from all testimonials, but your testimonials themselves are not affected.
