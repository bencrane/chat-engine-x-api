# How do I search or filter testimonials?

You want to use search for a testimonial. Don't worry, Senja makes it easy.

## Using search bar

1. Go to your Proof page: https://app.senja.io/
2. Type your search text in the bar (by name/text content...)

## Using filter

1. Go to your Proof page: https://app.senja.io/
2. Click the Filter button
3. Choose the filter parameter as you wish

> You can't filter testimonials by consent. The consent info is for your own note, you can click on each testimonial to see if your customer is giving public/private consent.
