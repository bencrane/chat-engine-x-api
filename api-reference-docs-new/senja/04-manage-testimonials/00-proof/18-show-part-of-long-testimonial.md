# How to show only part of a long testimonial

You can create a shorter version of a long testimonial by duplicating it and editing the copy to show only the most relevant parts.

## Steps to create a partial testimonial

1. **Duplicate the testimonial** - Find your long testimonial and create a copy of it
2. **Edit the duplicate** - Remove unnecessary sentences and keep only the key parts you want to display
3. **Add a unique tag** - Give the shortened version a specific tag to distinguish it from the original
4. **Use the tag for display** - Reference this tag when adding the testimonial to your landing page or widget

## Why use this method

This approach lets you keep both versions of the testimonial. You can use the shorter version for landing pages where space is limited, while keeping the full testimonial available for other widgets or contexts where more detail is appropriate.

The tagging system makes it easy to organize and find the right version of each testimonial when you need it.
