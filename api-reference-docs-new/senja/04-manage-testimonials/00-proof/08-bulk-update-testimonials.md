# Can you bulk update testimonials?

The Senja API does not currently support bulk updating testimonials.

## Can you update testimonials via the API?

The current Senja API does not support updating testimonials. You can retrieve and create testimonials using the API, but not modify existing ones.

## How can you bulk update testimonials?

If you need to perform a bulk update on your testimonials, such as changing their approval status, please contact our support team. Our team can assist you with internal processes to facilitate these changes.
