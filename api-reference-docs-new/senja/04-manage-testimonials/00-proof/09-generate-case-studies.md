# How to generate case studies from testimonials

You can easily transform any testimonial into a detailed case study using Senja's Magic Maker feature.

## Steps to generate a case study

1. Go to your Proof page: https://app.senja.io/
2. Navigate to the testimonial that you want
3. Click Magic Maker button (it will appear when you hover on the testimonial)
4. Choose Turn into case study
5. You'll be directed to the case study editing modal
6. You can Edit the case study, Add tag, or directly Save changes
7. When you're done, click Share to Copy share link or Export as PDF

The system will automatically generate a case study based on your testimonial content. This feature helps you create more detailed, story-driven content from your existing customer feedback.
