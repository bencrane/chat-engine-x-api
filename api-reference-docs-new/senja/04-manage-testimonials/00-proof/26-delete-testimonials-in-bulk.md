# How to delete testimonials in bulk

Bulk delete lets you remove multiple testimonials at once. This is permanent. Deleted testimonials can't be recovered.

> You need edit permissions to delete testimonials. This action is permanent and cannot be undone.

## Deleting testimonials

1. Go to your Proof page: https://app.senja.io/
2. There are 3 ways to select the testimonials:
   - Cherry pick manually
   - Using filter to select testimonials matching some criteria
   - Selecting all testimonials to remove everything
3. Once you've selected one or more testimonials, a floating bar appears at the bottom showing how many are selected
4. Click the red Delete button (trash icon)
5. A confirmation dialog will ask you to confirm
6. Click Delete to proceed, or Cancel to go back
7. Your selected testimonials will be permanently deleted

## Manual selection

Tick the check-box next to the testimonials you want to delete.

## Select using filter

1. Click the Filter button
2. Set the filter that you want
3. Tick the check-box next to the Filter button to select all filtered testimonials

## Select all testimonials

1. Tick the check-box next to the Filter button to select all testimonials
2. Click Delete on the popup toolbar in the bottom

> The select-all checkbox only selects testimonials currently loaded on screen. Scroll down to load more, then select those too if needed.
