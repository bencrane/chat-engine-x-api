# How do I find testimonials with public or private consent?

If you enable the Consent page in your form, the consent info will appear in your customer's testimonial.

## Finding the consent info

1. Go to your Proof page: https://app.senja.io/
2. Click on the testimonial you want to check
3. There will be 3 cases:
   - Consent page is off: no consent info available
   - Consent page is on, customer gives public consent
   - Consent page is on, customer gives private consent

## Marking consent type

The filtering in Proof page doesn't support filtering by consent. What you can do is:

1. Go to Proof page
2. Open the customer's testimonial
3. Add tag Public or Private

> You can also do the tagging in bulk, read How to add tags to multiple testimonials at once.
