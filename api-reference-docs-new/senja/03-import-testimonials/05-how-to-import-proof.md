# How to import your proof into Senja

One thing that makes Senja powerful is that it is the single source of truth for your proof. Your video testimonials, text testimonials, reviews, shout outs, screengrabs, and case studies all in one place.

## What proof can I import?

Senja supports the importing of the following types of proof:

- Text testimonials
- Video testimonials
- Screengrabs/images [of positive customer feedback]
- Case studies
- Reviews
- Social shoutouts

## What types of importing are there?

Choose the method that matches your source:

- How to auto-import testimonials from other platforms
- Import from web
- Can I import testimonials in bulk?
- Manual import
- Import testimonials with the Chrome extension

## What happens after the import?

Your testimonials will be stored in your Proof page: https://app.senja.io/

You can use these collected testimonials to create widget and share it.

If you're having trouble importing testimonials, read Troubleshooting Common Import Errors article.

## What do I do if my source isn't supported?

- You can suggest a new source, see below
- Alternatively import it as text or video using the Imports screen
- Or, import it using the free Senja Chrome Extension
