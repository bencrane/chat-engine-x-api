# How to import LinkedIn recommendation with Chrome extension

Import LinkedIn recommendations directly to your Senja dashboard using the Chrome extension. This works for recommendations on LinkedIn profile pages. Before you begin: Install the Senja Chrome extension from the Chrome Web Store and log into your LinkedIn account.

## Import recommendations from a profile

1. Go to your LinkedIn profile
2. Scroll down to the Recommendations section
3. Click the Senja extension icon in your Chrome toolbar to open the sidebar
4. Click the Import tab at the bottom of the sidebar
5. Make sure the Import Testimonial page is shown, on Text testimonial
6. Click the magic-cursor icon to grab the info from the page
7. Hover over the text/image you want
8. A suggestion block will appear
9. Click, and the info is grabbed
10. Scroll down, click Import testimonial button when you're done
11. Select LinkedIn from the dropdown menu

Use the full profile URL, not individual recommendation links. The extension only works on complete profile pages.

Your imported testimonials appear in the Proof page of your Senja dashboard, where you can review and approve them.
