# Import testimonials with the Chrome extension

Senja's Chrome extension lets you import your testimonials from anywhere on the web. Here's how to download it and import your first testimonial.

## Download the Extension

Download our free testimonial extension here.

## Import your first testimonial using the Chrome extension

In our example, we will import a testimonial from X.

1. Head to the testimonial you would like to import
2. Open the Senja Chrome extension
   - On your computer, open Chrome
   - At the top right, select Extensions (it's the puzzle icon)
   - Click on the Senja extension
3. Head to Import (located on the menu at the bottom of the extension sidebar)
4. Choose your platform (it will show up on the dropdown menu under Import Testimonial)

Can't find what you're looking for? The platform you're using may not be integrated with Senja yet. Head to step 6 below for instructions on capturing text and image testimonials instead.

5. Add a tag. Before importing testimonials, consider adding a tag. This will help you find and filter testimonials in the future.
6. Click Import. This button will look different depending on the platform you're importing from.
7. View your testimonial. Head to the Testimonials tab and ensure the testimonial imported correctly.

## Import text and image testimonials

If Senja isn't integrated with the platform hosting your testimonial, you can use the Text or Image option to grab them instead.

Here's how:

1. Choose your testimonial type. Click on the dropdown menu under Import Testimonial:
   - If your testimonial has text, choose Text testimonial
   - If your testimonial is an image, choose Image testimonial
2. Grab text or images:
   - Click the cursor button
   - Select the text and images you want to add to the corresponding fields
   - Watch it grab and fill for you

P.S. If you'd prefer to do this manually, you can type the info into the fields instead. But we highly recommend the grab tool for speed and accuracy.
