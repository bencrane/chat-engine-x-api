# Import LinkedIn post using Chrome extension

The Senja Chrome extension lets you import LinkedIn posts, comments, and recommendations directly from any LinkedIn page. Before you begin: Install the Senja Chrome extension and make sure you're logged into LinkedIn.

## Import a LinkedIn post

1. Navigate to the LinkedIn post you want to import in Chrome
2. Click the 3-dots
3. Click Import to Senja

The testimonial will appear in your dashboard on the Proof page, where you can review and approve it.

To import LinkedIn recommendations, navigate to the person's full profile page -- individual recommendation links won't work with the extension.
