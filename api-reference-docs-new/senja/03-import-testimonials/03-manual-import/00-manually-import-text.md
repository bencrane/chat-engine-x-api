# How can I manually import text testimonials?

If you have text testimonials from your customers, sent by email or other platforms, you can add the testimonials to your Senja dashboard using the Manual import.

## Manually import text testimonials

1. Go to Import > Manual import page: https://app.senja.io/import/manual
2. Choose Text testimonial
3. Complete the details:
   - **Customer name** (mandatory)
   - Tagline (you can put your customer's job title here)
   - Avatar
   - Email
   - Company
   - Team
   - Company Logo
   - Rating
   - Testimonial Title
   - Testimonial (here's where you put the text testimonial)
   - Testimonial Source
   - Attach images
   - Post URL
   - Date (add the date when you receive the testimonial the first time)
   - Add a tag
4. Once you're done, click Import testimonial
