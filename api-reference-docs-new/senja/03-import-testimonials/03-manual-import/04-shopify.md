# Import testimonials from Shopify

Senja lets you import product reviews from Shopify to use as testimonials and social proof. This feature is available on all plans. You can import Shopify reviews using manual import or the Chrome Extension.

## What you can do

- Manually import individual Shopify reviews
- Bulk import using the Senja Chrome Extension

## How to import the testimonials

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose Shopify.

Your Shopify review is now saved in Senja.

### Senja Chrome extension

You can use Senja Chrome extension to quickly grab multiple reviews from your Shopify store. Follow this guide: Import testimonials with the Chrome Extension.

The Chrome Extension makes it easy to bulk import reviews directly from your Shopify product pages.

Need help? If you run into any issues importing from Shopify, contact Senja Support and we'll help you get set up.
