# Import testimonials from WhatsApp

Senja lets you import conversations from WhatsApp to use as testimonials and social proof. This feature is available on all plans. You can import WhatsApp messages using manual import or the Chrome Extension.

## What you can do

- Manually import individual WhatsApp messages
- Bulk import using the Senja Chrome Extension

## How to import testimonials

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose WhatsApp.

Your WhatsApp message is now saved in Senja.

### Senja Chrome extension

1. Visit WhatsApp Web in your browser
2. Use the grabber tool in Senja Chrome extension to select individual message elements
3. Follow this guide: Import testimonials with the Chrome Extension

The Chrome Extension is great for importing multiple messages at once from WhatsApp Web.

Need help? If you run into any issues importing from WhatsApp, contact Senja Support and we'll help you get set up.
