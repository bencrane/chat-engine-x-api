# Import testimonials from Discord

Discord doesn't support auto-syncing, but you can manually import posts as testimonials in Senja. Before you begin: Copy the testimonial details from the Discord post you want to import.

**Account type:** All plans.

## How to import the testimonials

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose Discord.

Your testimonial appears in your dashboard tagged with Discord as the source.

### Senja Chrome extension

You can use the Senja Chrome Extension to quickly grab multiple messages from Discord. Follow this guide: Import testimonials with the Chrome Extension.

Need help? If you run into any issues, contact Senja Support and we'll help you get set up.
