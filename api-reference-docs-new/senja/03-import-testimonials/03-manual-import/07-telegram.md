# Import testimonials from Telegram

Senja lets you import individual posts from Telegram to use as testimonials and social proof. This feature is available on all plans. Auto-sync is not currently available for Telegram.

## What you can do

- Import individual posts from Telegram using manual import

## How to import testimonials

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose Telegram.

Your Telegram message is now saved in Senja.

### Senja Chrome extension

1. Visit Telegram Web in your browser
2. Use the grabber tool in Senja Chrome extension to select individual message elements
3. Follow this guide: Import testimonials with the Chrome Extension

The Chrome Extension is great for importing multiple messages at once from Telegram Web.

Need help? If you run into any issues importing from Telegram, contact Senja Support and we'll help you get set up.
