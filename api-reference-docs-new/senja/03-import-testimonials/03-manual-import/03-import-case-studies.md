# How to import case studies

You can import case studies into Senja so they are available to share from your account and the Chrome extension. Currently case studies are added as URLs.

## Manually import case studies

1. Go to Import > Manual import page: https://app.senja.io/import/manual
2. Choose Case Study
3. Complete the details:
   - **Customer name** (mandatory)
   - Tagline (you can put your customer's job title here)
   - Avatar
   - Email
   - Company
   - Team
   - Company Logo
   - **Case study URL** (mandatory)
4. Once you're done, click Import case study

The case study will be imported to your Proof page.
