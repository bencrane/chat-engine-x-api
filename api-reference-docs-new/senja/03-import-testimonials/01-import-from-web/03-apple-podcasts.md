# Import testimonials from Apple Podcasts

Senja lets you import reviews from your Apple Podcasts page to use as social proof. This feature is available on all plans. Auto-sync is not currently available for Apple Podcasts.

## What you can do

- Import all reviews from your Apple Podcasts page at once
- Select individual reviews to import specific testimonials

## How to import from Apple Podcasts

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Apple Podcasts icon
3. Paste your link under Apple Podcasts URL
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your Apple Podcasts reviews are now in Senja and ready to share.

## Other methods

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose Apple Podcasts.

### Senja Chrome extension

You can use Senja Chrome extension following this: Import testimonials with the Chrome Extension.

Need help? If you run into any issues importing your Apple Podcasts reviews, contact Senja Support and we'll help you get set up.
