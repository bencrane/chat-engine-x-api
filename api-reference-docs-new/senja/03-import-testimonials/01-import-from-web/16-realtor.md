# Import testimonials from Realtor

Senja lets you import testimonials from your Realtor page and keep them in one place. This feature is available on all plans. Auto-sync availability depends on your plan.

## What you can do

- Import all testimonials from your Realtor page at once
- Select individual testimonials to import specific entries
- Auto-sync new testimonials if available for Realtor (paid plans only)

## How to import from Realtor

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Realtor icon
3. Paste your link in Realtor profile URL field
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your Realtor testimonials are now in Senja and ready to share on your website.
