# Import testimonials from Twitter/X

Import a testimonial from a Twitter/X post into Senja. Before you begin: You'll need the URL of a public tweet you want to import. In Senja, this import option is labeled Twitter.

## What you can do

- Import a single public tweet
- Paste a tweet URL to pull the content into Senja
- Bulk imports and auto-sync aren't supported

## How to import from Twitter/X posts

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Twitter icon
3. Paste the link under Twitter tweet URL
4. Click Import testimonials

Your tweet is now imported into Senja.

## Other methods

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose Twitter.

### Senja Chrome extension

You can use Senja Chrome extension following this: Import testimonials with the Chrome Extension.

Need help? If you run into any issues importing from Twitter/X, contact Senja Support and we'll help you get set up.
