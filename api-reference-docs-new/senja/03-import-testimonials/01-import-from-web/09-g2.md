# Import testimonials from G2

Senja lets you import reviews from your G2 product page and automatically sync new ones as they come in. This feature is available on all plans. Auto-sync requires a paid plan.

## What you can do

- Import all reviews from your G2 page at once
- Select individual reviews to import specific testimonials
- Auto-sync to automatically pull in new reviews (paid plans only)

## How to import from G2

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the G2 icon
3. Paste your link under G2 product URL
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your G2 reviews are now in Senja and ready to share on your website.
