# Import testimonials from Skillshare

Senja lets you import testimonials from your Skillshare page and keep them in one place. This feature is available on all plans. Auto-sync availability depends on your plan.

## What you can do

- Import all testimonials from your Skillshare page at once
- Select individual testimonials to import specific entries
- Auto-sync new testimonials if available for Skillshare (paid plans only)

## How to import from Skillshare

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Skillshare icon
3. Paste your link in Skillshare class URL field
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your Skillshare testimonials are now in Senja and ready to share on your website.
