# Import testimonials from WordPress

Senja lets you import testimonials from your WordPress page and keep them in one place. This feature is available on all plans. Auto-sync availability depends on your plan.

## What you can do

- Import all testimonials from your WordPress page at once
- Select individual testimonials to import specific entries
- Auto-sync new testimonials if available for WordPress (paid plans only)

## How to import from WordPress

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the WordPress icon
3. Paste your link in WordPress plugin URL field
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your WordPress testimonials are now in Senja and ready to share on your website.
