# Import testimonials from Amazon

Senja lets you import product reviews from Amazon and automatically sync new ones as they come in. This feature is available on all plans. Auto-sync requires a paid plan.

## What you can do

- Import reviews from your Amazon product listing
- Auto-sync to automatically pull in new reviews (paid plans only)

## How to import from Amazon

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Amazon icon
3. Paste your link under Amazon product URL
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your Amazon reviews are now in Senja and ready to display.
