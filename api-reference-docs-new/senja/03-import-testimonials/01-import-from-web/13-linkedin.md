# Import testimonials from LinkedIn

Import LinkedIn posts, comments, and recommendations into Senja. Before you begin: To import LinkedIn Recommendations, use your full profile URL. Links to individual recommendations won't work.

## What you can do

- Import a single LinkedIn post
- Import LinkedIn comments
- Import LinkedIn Recommendations from your profile
- Auto-sync isn't available

## How to import from LinkedIn

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the LinkedIn icon
3. Choose what you want to import: Import post or Import recommendations
4. Paste your link under LinkedIn profile/post URL
5. Click Import testimonials
6. Choose which testimonials you want to import, or select all to import everything

Your LinkedIn testimonials are now in Senja and ready to review.

## Other methods

### Manual import

You can do manual import following this guide: How can I manually import text testimonials? In the Testimonial Source dropdown, choose LinkedIn.

### Direct integration

1. Open the LinkedIn post you want to import
2. Click the 3-dots
3. Click Import to Senja

### Senja Chrome extension

You can use Senja Chrome extension following this: Import testimonials with the Chrome Extension.

Need help? If you run into any issues importing from LinkedIn, contact Senja Support and we'll help you get set up.
