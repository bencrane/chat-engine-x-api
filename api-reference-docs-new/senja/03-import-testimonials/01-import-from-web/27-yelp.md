# Import testimonials from Yelp

Senja lets you import reviews from your Yelp business page and automatically sync new ones as they come in. This feature is available on all plans. Auto-sync requires a paid plan.

## What you can do

- Import all reviews from your Yelp page at once
- Select individual reviews to import specific testimonials
- Auto-sync to automatically pull in new reviews (paid plans only)

## How to import from Yelp

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Yelp icon
3. Paste your link under Yelp business URL
4. Click Import testimonials
5. Choose which testimonials you want to import, or select all to import everything

Your Yelp reviews are now in Senja and ready to share.
