# Import testimonials from Chrome web store

Use the Chrome Web Store importer to pull reviews from your extension listing into Senja. This feature is available on all plans. Imports are limited by your testimonial cap.

## How to import from Chrome Web Store

1. Go to Import > Import from web: https://app.senja.io/import/web
2. Click the Chrome Web Store icon
3. Paste your public Chrome Web Store URL

Use a public Chrome Web Store listing URL. Private or invalid pages won't return reviews.

4. Click Import testimonials
5. Select the reviews you want to import, or select all to import everything

Your Chrome Web Store reviews are now in Senja and ready to share.
