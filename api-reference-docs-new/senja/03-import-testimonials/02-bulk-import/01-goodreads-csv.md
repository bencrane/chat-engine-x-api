# How to import Goodreads reviews into Senja using CSV

## Introduction

You can import your Goodreads reviews into Senja using a CSV export and import process.

## Import Goodreads reviews

Senja does not currently offer a direct feature to import reviews from Goodreads. However, you can use a workaround to bring your reviews into Senja.

### Export from Goodreads

First, export your books from Goodreads. This export should include your reviews. For detailed instructions, refer to the Goodreads help article on exporting your reviews.

### Import into Senja via CSV

After exporting your reviews from Goodreads, you can import them into Senja using our CSV import feature. Learn how to perform a bulk import with CSV or XLS by visiting our bulk import guide.

## Suggest a feature

If you would like to see a direct import feature for Goodreads, you can add your idea to our feedback board. You will receive notifications when this feature is added.
