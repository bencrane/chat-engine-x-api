# Can I Adjust the Timing of Automated Follow-Up Emails?

This article explains how automated follow-up emails work in Senja and addresses timing adjustments.

## Automated follow-up emails

Senja uses automated follow-up emails for various purposes. These include sending thank you gifts after testimonial collection. You can also integrate with tools like Zapier for custom automations.

## Adjusting timing

Currently, you cannot change the timing of automated follow-up emails. The system sends these emails at a pre-determined interval. This feature is not configurable at this time.

## Requesting new features

We value your feedback for improving Senja. If you want to adjust email timing, please submit your idea. Visit our feedback portal at https://feedback.senja.io. Other users can upvote your suggestion there. You will receive notifications about feature updates.
