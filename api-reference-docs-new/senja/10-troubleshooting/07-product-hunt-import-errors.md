# Troubleshooting Product Hunt Import Errors

Having trouble importing testimonials from Product Hunt?

You're not alone.

This guide covers the most common Product Hunt import errors and provides step-by-step solutions to get your testimonials imported successfully.URL Format RequirementsThe most common cause of Product Hunt import failures is using the wrong URL format.

Product Hunt has different URL structures for products and launches:For Product ReviewsUse this format: https://www.producthunt.com/products/[product-name]Example: https://www.producthunt.com/products/followup-by-assetmuleFor Launch CommentsUse this format: https://www.producthunt.com/products/[product-name]/launches/[launch-name]Example: https://www.producthunt.com/products/followup-by-assetmule/launches/followup-by-assetmule-2Common Import ErrorsWrong URL Format ErrorIf you see an error about incorrect URL format, double-check that:You're using a Product Hunt URL (not Fiverr, personal sites, or other platforms)The URL follows the correct format shown aboveThere are no extra parameters or tracking codes in the URL"Contact Support" Error MessageIf you get a generic error asking you to contact support, try these steps:Verify you're using the correct Product Hunt URL formatTry a different browser or clear your browser cacheWait a few minutes and try again (temporary API issues can occur)Use the Chrome Extension as a fallback methodProduct-Specific Import FailuresSome Product Hunt pages may have technical issues that prevent importing.

If a specific product URL consistently fails:Try the basic product URL without any extra pathsUse the Chrome Extension method insteadCheck if the product page loads correctly in your browserStep-by-Step Troubleshooting
1.

Verify Your URLBefore importing, make sure your URL is correct:Go to Product Hunt and navigate to the product pageCopy the URL from your browser's address barEnsure it matches the format requirements above
2.

Try the Standard Import ProcessGo to https://app.senja.io/Click on Import testimonialsSelect "Import from Web"Choose Product Hunt and paste your URLClick "Import testimonials"
3.

If Import Fails, Use Chrome ExtensionThe Senja Chrome Extension is an excellent fallback when the standard import process fails:Install the Senja Chrome ExtensionNavigate to the Product Hunt page with the testimonials you wantOpen the Senja extensionChoose "Product Hunt" from the platform dropdownAdd any tags if neededClick "Import" The Chrome Extension works by capturing testimonials directly from the page, bypassing potential API issues that might affect the web import.

Browser and Device ConsiderationsSome import issues can be browser-specific:Try using Chrome, Firefox, or SafariDisable browser extensions that might interfereClear your browser cache and cookiesMake sure JavaScript is enabledWhen to Contact SupportContact our support team if:You've tried all troubleshooting steps and still can't importYou're getting consistent errors with multiple Product Hunt URLsThe Chrome Extension import also failsYou need help with bulk importing from Product HuntWhen contacting support, please include:The exact Product Hunt URL you're trying to importThe error message you're seeingYour browser and device informationScreenshots of the error (if possible) Most Product Hunt import issues can be resolved using the correct URL format or the Chrome Extension fallback method.

Don't let import errors stop you from showcasing your amazing testimonials!
