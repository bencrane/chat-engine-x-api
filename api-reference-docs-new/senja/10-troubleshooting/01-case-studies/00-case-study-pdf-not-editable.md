# My downloaded case study PDF isn't editable.

This article explains how to extract and edit text from your download case studyUnderstand scanned PDFsThe downloaded PDFs are image-based documents.

You cannot directly edit text within them.

To modify the content, you must first extract the text.

Extract text using online toolsMany free online tools can help you extract text.

These tools use Optical Character Recognition (OCR) technology. OCR converts the image-based text into editable digital text.

Edit your extracted textOnce the text is extracted, you can edit it.

You can then use this content for your testimonials.
