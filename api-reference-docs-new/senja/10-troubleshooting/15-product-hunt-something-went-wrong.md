# Resolving Product Hunt 'Something went wrong' Import Errors

OverviewThis guide will help you troubleshoot and resolve the "Something went wrong.

Contact us..." error that can occur when importing testimonials from Product Hunt.

The most common cause of this error is using an incorrect URL format. URL is key: Using the correct Product Hunt URL is critical for a successful import.

This guide will show you how to find the right one.

Why the Error OccursThe import tool is designed to work with specific Product Hunt page structures.

If you use a general product page or a reviews tab URL, the import will likely fail.

You need to use the URL of the specific launch page.

Finding the Correct Launch URLFollow these steps to locate the correct URL for your Product Hunt import:Navigate to the Product Hunt page for the product you want to import testimonials from.

From the product's main page, look for the "Launches" tab or a link to a specific launch.

In this example, you would click on the launch titled "The Influencer AI".

Once you are on the launch page, copy the URL directly from your browser's address bar.

Correct vs.

Incorrect URLsTo ensure a successful import, your URL should look like this: Correct URL format: https://www.producthunt.com/posts/[product-name]Example: https://www.producthunt.com/posts/the-influencer-aiAvoid using these incorrect URL formats: Incorrect URL formats:Product page: https://www.producthunt.com/products/[product-name]Reviews tab: https://www.producthunt.com/products/[product-name]/reviewsStep-by-Step Import GuideGo to the Import testimonials page in your Senja account.

Select "Import from Web" and choose Product Hunt.

Paste the correct launch URL you copied.

Click "Import testimonials".

If you continue to experience issues, our Chrome Extension is a reliable alternative for importing testimonials directly from the Product Hunt page.

When to Contact SupportIf you've followed all the steps above and are still encountering the error, please contact our support team.

When you do, please provide the following information:The exact Product Hunt URL you are trying to import from.A screenshot of the error message.

The browser and device you are using.
