# Troubleshooting testimonial import errors

If you're encountering errors when trying to import testimonials, this guide will help you troubleshoot and resolve common issues.

Symptom: "Something went wrong" errorThis is a generic error message that can be caused by a few different things.

Here's how to troubleshoot it:
1.

Check the import URLSome import sources require a specific URL format.

For example, when importing from Product Hunt, you need to use the URL of a specific launch, not the general product page.

Incorrect: https://www.producthunt.com/products/the-influencer-aiCorrect: https://www.producthunt.com/products/the-influencer-ai/launches/the-influencer-ai
2.

Platform-specific bugsOccasionally, there might be a bug with a specific import source.

We work to resolve these as quickly as possible.

If you suspect this is the case, please contact us and we will investigate.3.

Manual import as a workaroundIf you're still unable to import your testimonials, you can always manually import them.

This allows you to add testimonials one by one, giving you full control over the content.

Symptom: Testimonials are not appearing after importif you've successfully imported your testimonials but they are not appearing on your Wall of Love or in your widgets, it may be due to the following:
1.

Testimonials need to be approvedBy default, all imported testimonials need to be approved before they are displayed.

You can approve testimonials from your dashboard.

You can disable this behaviour by heading to your settings page and toggling off "Approval required".

Still having issues?

If you have tried all the steps above and are still having trouble, please contact us.

We're here to help!
