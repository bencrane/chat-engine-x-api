The `@remotion/fonts` package provides an API for [loading local fonts](/docs/fonts) or remote fonts from a URL into a Remotion component.

## Installation[​](#installation)

- Remotion CLI
- npm
- bun
- pnpm
- yarn

```

npx remotion add @remotion/fontsCopy
```

```

npm i --save-exact @remotion/[[email protected]](/cdn-cgi/l/email-protection)Copy
```
This assumes you are currently using v4.0.441 of Remotion.
Also update `remotion` and all ``@remotion/*`` packages to the same version.
Remove all `^` character in front of the version numbers of it as it can lead to a version conflict.

