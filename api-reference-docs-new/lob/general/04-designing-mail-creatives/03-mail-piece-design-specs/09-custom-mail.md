# Custom mail

As a direct mail automation platform, Lob's focus is to offer [set mail formats](https://help.lob.com/print-and-mail/designing-mail-creatives/mail-piece-design-specs) that can be programmatically sent at scale. These are postcards, self-mailers, snap packs, letters, and checks. However, for customers who need to send greater mail variations beyond these formats, we are open to exploring options with you.&#x20;

Please reach out to your Account Manager for more information on program eligibility and requirements.
