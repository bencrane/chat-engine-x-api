# SDKs and Tools

Lob provides official SDKs for popular programming languages to help you integrate with the Lob API.

## Official SDKs

- **Python**: [lob-python](https://github.com/lob/lob-python)
- **Node.js**: [lob-node](https://github.com/lob/lob-node)
- **Ruby**: [lob-ruby](https://github.com/lob/lob-ruby)
- **PHP**: [lob-php](https://github.com/lob/lob-php)
- **Java**: [lob-java](https://github.com/lob/lob-java)

## Installation

Each SDK can be installed via its respective package manager. See the individual SDK repositories for detailed installation instructions.
