# TheirStack - General - Introduction

Access to the largest database of jobs and technographics.

Welcome to the TheirStack API - the largest database of jobs and technographics.

You can download the OpenAPI specification as a JSON file or YAML file.

## Overview

**Job Search**
Search and filter real-time and historical job postings from more than 100 countries with flexible query parameters.

**Company Search**
Find companies by the technology they use or by the roles they are hiring for.

**Technographics**
Get the list of technologies used by any company in our database.
