# Reverse ETL Developer Guides

* * *

  *  __less than a minute

  * 


This section contains the following Reverse ETL developer guides:

Feature| Description  
---|---  
[Set Up Reverse ETL Connection](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/retl-connection-setup/>)| Connect your Reverse ETL sources to destinations in RudderStack.  
[Trigger Syncs from dbt Cloud](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/trigger-syncs-from-dbt-cloud/>)| Trigger Reverse ETL syncs when dbt Cloud jobs complete using webhooks.  
[JSON Data Mapping](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/json-data-mapping/>)| Specify data mappings for Reverse ETL connections via JSON.  
[Visual Data Mapping](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/visual-data-mapper/>)| Map your warehouse columns to destination tables with RudderStack’s Visual Data Mapper functionality.  
[Start and Stop Syncs](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/start-stop-syncs/>)| Start, stop, and trigger a Reverse ETL sync manually.  
[Sync Modes](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/sync-modes/>)| Learn about different sync modes to send data from Reverse ETL sources to your destinations.  
[Cursor Column Support](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/cursor-column-support/>)| Run cost-effective SQL queries using cursor columns for efficient incremental upsert syncs.  
[Sync Observability Settings](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/sync-observability/>)| Options to store connection level sync logs and retry sending failed records.  
[Sync Schedule Settings](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/sync-schedule-settings/>)| Options to sync data from your Reverse ETL sources on a specified schedule.  
[FAQ](<https://www.rudderstack.com/docs/data-pipelines/reverse-etl/developer-guides/faq/>)| Answers to some of the commonly asked questions on Reverse ETL.
