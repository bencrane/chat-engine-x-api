# Data Apps

Run RudderStack Data Apps on top of your Customer 360 to build attribution, propensity scores, and personalization projects.

* * *

  * __less than a minute

  * 


RudderStack **Data Apps** help data teams accelerate key projects like attribution, propensity scoring, and real-time personalization.

You can leverage the power of your Customer 360 - without the need for complex SQL or Python coding - and enable your data teams to deliver impactful results quickly.

The Data Apps cater to the following use cases:

  * **[Attribution](<https://www.rudderstack.com/docs/archive/profiles/0.17/data-apps/attribution/>)** : Deliver accurate, report-ready first and last-touch attribution data for paid campaigns, including RoAS.
  * **[Propensity scores](<https://www.rudderstack.com/docs/archive/profiles/0.17/data-apps/propensity/>)** : Generate propensity scores using your own churn or conversion definitions, then automatically sync them to user profiles in business tools.
  * **[Real-time personalization](<https://www.rudderstack.com/docs/archive/profiles/0.17/data-apps/real-time-personalization/>)** : Make your customer 360 available via API to drive real-time website and app personalization, enhancing user engagement, and conversion rates.
