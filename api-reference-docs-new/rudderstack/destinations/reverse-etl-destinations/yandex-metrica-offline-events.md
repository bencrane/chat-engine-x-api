# Yandex.Metrica Offline Events Destination Beta

Send your offline conversions data from RudderStack to Yandex.Metrica.

* * *

  * __less than a minute

  * 


[Yandex.Metrica](<https://metrica.yandex.com/>) is a web analytics service. You can leverage this integration to upload offline events to Yandex.Metrica for effective offline conversion tracking.
