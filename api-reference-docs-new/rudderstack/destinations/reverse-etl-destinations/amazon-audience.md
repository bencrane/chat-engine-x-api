# Amazon Audience Beta

Send your event data from RudderStack to Amazon Audience.

* * *

  * __less than a minute

  * 


[Amazon Audience](<https://advertising.amazon.com/insights-and-planning/audiences>) is a targeting solution that allows advertisers to reach specific customer segments based on Amazon’s first-party data, including browsing and purchase behavior.
