# LaunchDarkly Segments Beta

Send your event data from RudderStack to LaunchDarkly Segments.

* * *

  * __less than a minute

  * 


[LaunchDarkly](<https://launchdarkly.com/>) is a feature flag management platform. LaunchDarkly members use [Segments](<https://docs.launchdarkly.com/home/segments>) to perform targeted rollouts of features.

You can use the **LaunchDarkly Segments** destination to sync your audiences from your warehouse to LaunchDarkly.
