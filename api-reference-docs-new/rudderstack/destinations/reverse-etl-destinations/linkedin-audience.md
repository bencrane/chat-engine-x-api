# LinkedIn Audience Beta

Send your event data from RudderStack to LinkedIn Audience.

* * *

  * __less than a minute

  * 


[LinkedIn](<https://www.linkedin.com/>) offers a centralized Data Management Platform (DMP) that aggregates LinkedIn advertiser data and empowers B2B marketers to identify and reach their target audiences across different platforms and ad exchanges. It also maps the user information from LinkedIn advertisers and member profiles to output a custom audience list in a format that you can use for targeted advertising.
