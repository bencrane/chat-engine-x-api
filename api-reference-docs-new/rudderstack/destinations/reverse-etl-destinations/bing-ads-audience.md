## FAQ

#### What is a custom audience?

A custom audience is a remarketing list generated using your customer data to create rich user segments. You can use these segments in Microsoft Advertising for search remarketing, customer retargeting, and various other use cases.

Read more about custom audiences in [Microsoft’s documentation](<https://help.ads.microsoft.com/#apex/3/en/56849/0>).
