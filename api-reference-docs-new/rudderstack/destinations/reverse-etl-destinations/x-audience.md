# X Audience (formerly Twitter Audience) Beta

Send your event data from RudderStack to X Audience.

* * *

  * __less than a minute

  * 


[X Audience](<https://business.x.com/en/advertising.html>) allows advertisers to extend their advertising campaigns via X for better reach and visibility.

RudderStack supports sending data to X Audience from your [Reverse ETL source](<https://www.rudderstack.com/docs/sources/reverse-etl/>).
