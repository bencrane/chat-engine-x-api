# Singular Destination

Send your event data from RudderStack to Singular.

* * *

  * __less than a minute

  * 


[Singular](<https://www.singular.net/>) is a marketing intelligence platform that lets you transform your marketing data into accurate, granular, and actionable insights that drive growth.
