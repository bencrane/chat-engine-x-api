# Smartly Destination Beta

Send your event data from RudderStack to Smartly.

* * *

  * __less than a minute

  * 


[Smartly](<https://www.smartly.io/>) is a platform that automates and optimizes social media advertising. It enables businesses to create, manage, and scale campaigns across various channels like Facebook, Instagram, and TikTok, using data-driven insights for better performance.
