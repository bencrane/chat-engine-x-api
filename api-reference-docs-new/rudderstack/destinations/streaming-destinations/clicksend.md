# ClickSend Destination Beta

Send your event data from RudderStack to ClickSend.

* * *

  * __less than a minute

[ClickSend](<https://www.clicksend.com/in/>) is a cloud-based service that allows businesses to send and receive SMS, email, voice, fax, and letters through APIs, enabling efficient communication with customers and clients.
