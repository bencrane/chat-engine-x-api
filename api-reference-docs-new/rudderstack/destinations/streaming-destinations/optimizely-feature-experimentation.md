# Optimizely Feature Experimentation

Send your event data from RudderStack to Optimizely Feature Experimentation.

* * *

  * __less than a minute

  * 


[Optimizely Feature Experimentation](<https://www.optimizely.com/products/full-stack/>) (formerly Optimizely Full Stack) is a powerful A/B testing, feature flagging and experimentation platform. It lets you track every aspect of your customer’s overall product journey, and experiment with various application features.
