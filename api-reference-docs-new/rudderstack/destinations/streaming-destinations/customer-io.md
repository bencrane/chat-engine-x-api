# Customer.io Destination

Send your event data from RudderStack to Customer.io.

* * *

  * __less than a minute

  * 


[Customer.io](<https://customer.io/>) is a popular marketing platform for sending targeted emails and push and SMS notifications to improve customer engagement and overall conversion rate.
