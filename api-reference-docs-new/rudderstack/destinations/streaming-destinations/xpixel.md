# X Pixel (Twitter Pixel) Destination Beta

Send your event data from RudderStack to X Pixel.

* * *

  * __less than a minute

  * 


[X Pixel](<https://business.x.com/en/help/campaign-measurement-and-analytics/conversion-tracking-for-websites.html>) provides advanced analytics and tracking tools for websites and apps. It enables businesses to monitor user interactions, optimize marketing efforts, and gain insights into customer behavior through data-driven decisions.
