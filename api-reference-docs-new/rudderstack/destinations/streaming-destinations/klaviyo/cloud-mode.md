# Klaviyo Cloud Mode Integration

Send events to Klaviyo in RudderStack cloud mode.

* * *

  * __less than a minute

  * 


RudderStack supports the following Klaviyo APIs to send your event data in [cloud mode](<https://www.rudderstack.com/docs/destinations/rudderstack-connection-modes/#cloud-mode>):

  * [API v2024-10-15](<https://www.rudderstack.com/docs/destinations/streaming-destinations/klaviyo/cloud-mode/new-api/>)
  * [API v2023-02-22](<https://www.rudderstack.com/docs/destinations/streaming-destinations/klaviyo/cloud-mode/old-api/>) (**deprecated**)
