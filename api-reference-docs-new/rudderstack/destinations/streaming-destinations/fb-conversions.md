# Facebook Conversions

Send your event data from RudderStack to Facebook using the Conversions API.

* * *

  * __less than a minute

  * 


With Facebook’s [Conversions API](<https://developers.facebook.com/docs/marketing-api/conversions-api/>), you can send server-side events to Facebook and use them for your measurement, reporting, and optimization use-cases.
