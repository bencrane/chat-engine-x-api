# HubSpot Cloud Mode Integration

Send events to HubSpot using the RudderStack cloud mode.

* * *

  * __less than a minute

  * 


RudderStack lets you send your event data to HubSpot via [cloud mode](<https://www.rudderstack.com/docs/destinations/rudderstack-connection-modes/#cloud-mode>) using the HubSpot [new](<https://developers.hubspot.com/docs/api/overview>) and [legacy](<https://legacydocs.hubspot.com/docs/overview?_ga=2.34803302.670362313.1663315856-97001172.1658910392>) (deprecated soon) API.

Find the open source code for this destination in the [GitHub repository](<https://github.com/rudderlabs/rudder-transformer/tree/master/src/v0/destinations/hs>).
