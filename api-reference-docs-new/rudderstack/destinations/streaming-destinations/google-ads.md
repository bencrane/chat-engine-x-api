# Google Ads (gtag.js) Destination

Send your event data from RudderStack to Google Ads.

* * *

  * __less than a minute

  * 


> ![tip](/docs/images/tip.svg)
> 
> **Tip:** **Choose the right Google Ads destination**
> 
>   * Use this destination to send events in **web device mode**. To enable enhanced conversions, turn on the **Allow identify calls for the Google Ads destination** [dashboard setting](<https://www.rudderstack.com/docs/destinations/streaming-destinations/google-ads/setup-guide/#connection-settings>) and send `identify` calls with your user data (for example, email and phone).
>   * Use the [Google Ads Enhanced Conversions](<https://www.rudderstack.com/docs/destinations/streaming-destinations/google-adwords-enhanced-conversions/>) destination to send events in **cloud mode** (server-to-server integration) using the Google Ads API.
> 


[Google Ads](<https://ads.google.com/>) is Google’s premier online advertising platform. It can be used for efficient marketing strategies like product listing, service offerings, and conversion tracking.
