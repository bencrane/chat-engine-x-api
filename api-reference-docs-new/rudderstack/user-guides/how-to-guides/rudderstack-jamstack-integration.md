# Integrate RudderStack with Jamstack Site

Add RudderStack to your Jamstack site.

* * *

  * __less than a minute

  * 


RudderStack supports integration with various popular [Jamstack](<https://jamstack.org/>) frameworks like Gatsby, Angular, Vue, Next.js, Hugo, Jekyll, and more.

You can now seamlessly integrate the [RudderStack JavaScript SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-javascript-sdk/>) into your web app/site built on these frameworks and set up your event streams in no time.
