# watchOS

Use the RudderStack iOS (Obj-C) SDK for watchOS.

* * *

  * __less than a minute

  * 


[watchOS](<https://developer.apple.com/watchos/>) is the operating system developed for Apple Watch, and is based on iOS. watchOS provides an also lets you leverage the watchOS SDK to develop apps that deliver interactive user experiences.

## iOS (Obj-C) SDK support for watchOS

Starting from version **1.3.0** , the [RudderStack iOS (Obj-C) SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/>) supports the watchOS platform.

> ![success](/docs/images/tick.svg)
> 
> You can now integrate the iOS (Obj-C) SDK with your watchOS apps and seamlessly track user events without the need for **any** additional configuration.

## Supported events

The watchOS platform supports all events that are supported by the [iOS (Obj-C) SDK](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/>).

  * [Track](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#track>)
  * [Screen](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#screen>)
  * [Identify](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#identify>)
  * [Group](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#group>)
  * [Alias](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#alias>)
  * [Reset](<https://www.rudderstack.com/docs/sources/event-streams/sdks/rudderstack-ios-sdk/#reset>)
