# RudderStack Shopify Source Solution

Ingest event data from Shopify into RudderStack.

* * *

  * __less than a minute

  * 


RudderStack’s comprehensive Shopify Source Solution is designed to support five different Shopify store setup scenarios. It consists of two different products and three different ways of tracking events.

The guides in this section outline all the scenarios and RudderStack’s recommended tracking solution for each of them.

## In this section
