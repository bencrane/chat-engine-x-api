# SDKs

Setup guides for integrating RudderStack SDKs with your web, mobile, and server-side platforms.

* * *

  * __less than a minute

  * 


RudderStack provides the following open source SDKs for your web, mobile, and server-side applications:

### Web

Track customer event data from your website and web apps and send it to the specified destinations.
