# List IP profiles

`GET /accounts/{account_id}/devices/ip-profiles`

Lists WARP Device IP profiles.

## Parameters

- **account_id** (string, required) [path]: 
- **per_page** (integer, optional) [query]: The number of IP profiles to return per page.
