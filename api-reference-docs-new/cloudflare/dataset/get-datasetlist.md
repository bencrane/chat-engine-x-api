# Lists all datasets in an account

`GET /accounts/{account_id}/cloudforce-one/events/dataset`



## Parameters

- **account_id** (string, required) [path]: Account ID.

## Response

### 200

Returns a list of dataset in an account.

Type: array

### 400

Bad Request.

- **result** (object):
