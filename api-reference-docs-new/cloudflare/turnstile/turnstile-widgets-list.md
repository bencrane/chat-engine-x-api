# List Turnstile Widgets

`GET /accounts/{account_id}/challenges/widgets`

Lists all turnstile widgets of an account.

## Response

### 200

List Turnstile Widgets

_Empty object_

### 4XX

List Turnstile Widgets Error

- **errors** (object): 
- **messages** (object): 
- **result** (object): 
- **success** (boolean): Whether the API call was successful
