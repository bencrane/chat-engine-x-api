# Get global schema validation settings

`GET /zones/{zone_id}/schema_validation/settings`

Retrieves the current global schema validation settings for a zone.

## Response

### 200

Success

_Empty object_

### 4XX

Failure

- **errors** (object): 
- **messages** (object): 
- **result** (object): 
- **success** (boolean): Whether the API call was successful.
