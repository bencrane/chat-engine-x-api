# Get a PAC file

`GET /accounts/{account_id}/gateway/pacfiles/{pacfile_id}`

Get a single Zero Trust Gateway PAC file.

## Parameters

- **pacfile_id** (string, required) [path]: 
- **account_id** (string, required) [path]: 

## Response

### 200

Returns a PAC file response.

_Empty object_

### 4XX

Returns a PAC file response failure.

_Empty object_
