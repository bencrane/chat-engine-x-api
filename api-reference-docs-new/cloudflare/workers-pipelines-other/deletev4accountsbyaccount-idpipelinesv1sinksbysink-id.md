# Delete Sink

`DELETE /accounts/{account_id}/pipelines/v1/sinks/{sink_id}`

Delete Pipeline in Account.

## Parameters

- **account_id** (string, required) [path]: 
- **sink_id** (string, required) [path]: 
- **force** (string, optional) [query]: 

## Response

### 200

Indicates a successfully deleted Sink.

### 4XX

Indicates an error in listing Sinks.
