# iOS Demo App

**Download & Resources:**
https://www.twilio.com/docs/conversations/sdk-download-install#ios-sdk

## Overview

A full-featured iOS Swift application demonstrating Twilio Conversations functionality:

- Create and join conversations
- Add participants to conversations
- Send and receive messages
- Media message support
- Push notifications
- Real-time updates

## Features Demonstrated

- Authentication with Twilio access tokens
- Conversation creation and management
- Real-time messaging
- Participant management
- APNs integration for push notifications
- Media attachments

## Requirements

- Xcode 13+
- iOS 12.0+
- Swift 5.0+
- Apple Developer account (for push notifications)
