# Android Demo App

**Download & Resources:**
https://www.twilio.com/docs/conversations/sdk-download-install#android-sdk

## Overview

A full-featured Android Kotlin application demonstrating Twilio Conversations functionality:

- Create and join conversations
- Add participants to conversations
- Send and receive messages
- Media message support
- Push notifications
- Real-time updates

## Features Demonstrated

- Authentication with Twilio access tokens
- Conversation creation and management
- Real-time messaging
- Participant management
- Firebase Cloud Messaging integration
- Media attachments

## Requirements

- Android Studio
- Android SDK 21+
- Kotlin
- Firebase project (for push notifications)
