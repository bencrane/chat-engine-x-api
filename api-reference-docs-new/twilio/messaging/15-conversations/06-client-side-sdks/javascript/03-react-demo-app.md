# React Demo App

**GitHub Repository:**
https://github.com/twilio/twilio-conversations-demo-react

## Overview

A full-featured React application demonstrating Twilio Conversations functionality:

- Create and join conversations
- Add participants to conversations
- Send and receive messages
- Media message support
- Real-time updates

## Getting Started

```bash
git clone https://github.com/twilio/twilio-conversations-demo-react.git
cd twilio-conversations-demo-react
npm install
npm start
```

## Features Demonstrated

- Authentication with Twilio access tokens
- Conversation creation and management
- Real-time messaging
- Participant management
- Push notifications
- Media attachments
