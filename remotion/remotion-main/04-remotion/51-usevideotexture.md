"---\nimage: /generated/articles-docs-use-video-texture.png\nid: use-video-texture\nsidebar_label: useVideoTexture()\ncrumb: '@remotion/three'\n---\n\n# ~useVideoTexture()~\n\n:::warning\nDeprecated - we recommend using [`@remotion/media`](/docs/videos/as-threejs-texture) instead.\n:::\n\nAllows you to use a video in React Three Fiber that is synchronized with Remotion's `useCurrentFrame()`.\n\nTo use a video in a Three.JS context, you first have to render it and assign it a ref. If you only want to use it in a React Three Fiber Scene, you can make it invisible by adding a `{position: \"absolute\", opacity: 0}` style.\n\n```tsx twoslash\nimport {useRef} from 'react';\nimport {Html5Video} from 'remotion';\nimport src from './vid.mp4';\n\nconst MyVideo = () => {\n  const videoRef = useRef<HTMLVideoElement | null>(null);\n\n  return <Html5Video ref={videoRef} src={src} style={{position: 'absolute', opacity: 0}} />;\n};\n```\n\nTo convert the video to a video texture, place the `useVideoTexture()` hook in the same component.\n\n```tsx twoslash\nimport React from 'react';\nconst videoRef: React.MutableRefObject<HTMLVideoElement | null> = React.useRef(null);\n// ---cut---\nimport {useVideoTexture} from '@remotion/three';\n\n// ...\n\nconst texture = useVideoTexture(videoRef);\n```\n\nThe return type of it is a `THREE.VideoTexture | null` which you can assign as a `map` to for example `meshBasicMaterial`. We recommend to only render the material when the texture is not `null` to prevent bugs.\n\n```tsx twoslash\nimport React from 'react';\nimport {useVideoTexture} from '@remotion/three';\nconst videoRef: React.MutableRefObject<HTMLVideoElement | null> = React.useRef(null);\nconst videoTexture = useVideoTexture(videoRef);\n// ---cut---\n{\n  videoTexture ? <meshBasicMaterial map={videoTexture} /> : null;\n}\n```\n\n## See also\n\n- [Source code for this function](https://github.com/remotion-dev/remotion/blob/main/packages/three/src/use-video-texture.ts)\n- [Example usage: React Three Fiber template](/templates/three)\n- [`<ThreeCanvas />`](/docs/three-canvas)\n"
