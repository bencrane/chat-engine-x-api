# ElevenLabs Custom Voice Integration

## Overview
This documentation explains how to connect a cloned ElevenLabs voice to Vapi. The process requires "an API subscription" to function.

## Setup Process

The integration involves four main steps:

1. **Subscribe to ElevenLabs** - Visit their pricing page and select an API plan matching your requirements.

2. **Obtain API credentials** - Access the Profile + Keys section on ElevenLabs to retrieve your API key.

3. **Configure in Vapi** - Go to the Integrations settings in your Vapi dashboard and enter your ElevenLabs API key. The system will automatically synchronize your voice library upon saving.

4. **Deploy your voice** - Locate your cloned voice through the voices tab on the assistants page and implement it with your assistant.
