# PlayHT Custom Voice Setup in Vapi

## Overview

The documentation outlines how to integrate a custom PlayHT voice with Vapi through a five-step process that requires an active API subscription.

## Key Steps Summary

The setup involves:

1. **Subscription**: Obtaining an API plan from PlayHT's pricing page
2. **Credentials**: Retrieving User ID and Secret Key from PlayHT's API Access section
3. **Integration**: Adding API credentials to Vapi's integration settings
4. **Synchronization**: Using the Voice Library to sync with PlayHT
5. **Implementation**: Locating and deploying the cloned voice with an assistant

## Important Note

"An API subscription is required for this process," making it essential to complete the first step before proceeding.
