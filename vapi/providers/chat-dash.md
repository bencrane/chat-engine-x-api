# ChatDash Integration with Vapi

ChatDash is a white-label dashboard platform for AI agent agencies that connects with Vapi. The integration enables agencies to access analytics, call logs, and billing functionality.

## Setup Process

Getting started requires minimal effort: "Simply input your Vapi Agent ID and API Key, and ChatDash instantly pulls in data to configure your dashboard." The platform then automatically configures with live performance metrics and call history.

## Core Features

The platform offers several key capabilities:

- **Real-time performance tracking** with immediate access to analytics and detailed call records from your Vapi agents
- **Professional branding options** allowing customization of logos, domains, and color schemes
- **Automated billing management** through Stripe integration for usage-based client invoicing

## Getting Started

The onboarding workflow involves entering your Vapi credentials, allowing ChatDash to automatically configure your dashboard, applying your branding elements, and then using the platform to "track performance, manage billing, and make data-driven decisions."

For implementation assistance, the support team can be reached at support@chatdash.com. A tutorial video demonstrating the integration process is available on their YouTube channel.
