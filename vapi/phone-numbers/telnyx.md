# Import Number from Telnyx

## Overview

The documentation explains how to integrate existing Telnyx phone numbers into the Vapi platform to enable outbound calling capabilities with assistants or squads.

## Initial Setup Steps

Users must complete three preliminary actions:

1. **Access Phone Numbers Section**: Navigate to the Phone Numbers tab within the Vapi dashboard
2. **Create New Number Entry**: Click the "Create a Phone Number" button
3. **Select Telnyx Provider**: Choose the Telnyx tab and input phone number information

A prerequisite note indicates that "You'll need to have an active Telnyx account with phone numbers that you want to import."

## Outbound Calling Configuration

To activate outbound calling functionality, users should:

1. **Portal Access**: Log into the Telnyx Portal
2. **Profile Setup**: Create or select an outbound voice profile
3. **Add Vapi Integration**: Include Vapi as a connection under the "Connections and Applications" section
4. **Finalize Settings**: Save all configuration changes

## Important Consideration

The guide emphasizes that proper Telnyx configuration is essential: "Without this configuration, outbound calling functionality will not work properly with your Telnyx numbers on the Vapi platform."
