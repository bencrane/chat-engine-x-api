# Vapi Enterprise

## Overview
Vapi Enterprise is designed for organizations building production applications, offering comprehensive support from initial concept through large-scale deployment.

## Key Features

The enterprise tier provides several capabilities:

- **Scalability**: "Unlimited concurrency and higher rate limits"
- **Infrastructure**: Reserved capacity on weekly deployment clusters
- **Support**: Hands-on technical assistance and shared Slack channel access
- **Compliance**: HIPAA BAA and SOC 2 (Type II) certifications
- **Authentication**: Single Sign On support for Okta, Azure AD, SAML, and OIDC
- **Access Management**: Role Based Access Control functionality
- **Reliability**: Service level agreement commitments

These features enable organizations to deploy voice applications with enterprise-grade infrastructure, security, and support.
